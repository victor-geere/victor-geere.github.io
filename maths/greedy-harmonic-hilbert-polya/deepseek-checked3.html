What is proved: The greedy harmonic decomposition converges at rate O(1/N). The correlation kernel K(n,m) has a closed rational form. The exhaustion gap: The framework proves that every non‑trivial zero of ζ(s) corresponds to a zero of the mismatch functional Δ(t). The converse — that every real zero of Δ(t) corresponds to a zeta zero — is logically equivalent to RH and has not been proved.

1. Greedy Harmonic Decomposition

1.1 Harmonic angles

\[ \alpha_n = \frac{\pi}{n+2}, \qquad n = 0,1,2,\dots \] \[ \sum_{n=0}^\infty \alpha_n = \infty, \qquad \sum_{n=0}^N \alpha_n = \pi\bigl(H_{N+2}-1\bigr), \] where \(H_k = \sum_{j=1}^k 1/j\).

1.2 Greedy expansion of an angle

For a target angle \(\theta\in[0,\pi]\) set \(\theta_0=0\) and define

\[ \delta_n(\theta) = \begin{cases} 1, & \text{if } \theta - \theta_n \geq \alpha_n,\\[2pt] 0, & \text{otherwise}, \end{cases} \qquad \theta_{n+1} = \theta_n + \delta_n(\theta)\,\alpha_n . \]

The expansion converges at rate \(O(1/N)\):

\[ \theta = \sum_{n=0}^\infty \delta_n(\theta)\,\alpha_n, \qquad |\theta - \theta_N| \leq \frac{\pi}{N+2}. \]

1.3 Geometric fusion rule

\[ s_{n+1} = s_n\sqrt{1-h_n^2} + \delta_n\!\left(h_n\sqrt{1-s_n^2} + s_n\sqrt{1-h_n^2}\right), \qquad h_n \approx \sin(\alpha_n). \]

1.4 Selection density

\[ D(N,\theta) := \sum_{n=0}^N \delta_n(\theta) = \frac{\theta}{\pi}\ln(N+2) + O(1). \]

■ Proved. Follows from the greedy rule and the asymptotics of harmonic numbers.

1.5 Python simulation

import math

def harmonic_sine_approx(theta, max_n=10000):
    s = 0.0
    accumulated = 0.0
    for n in range(max_n + 1):
        alpha = math.pi / (n + 2)
        delta = 1 if (theta - accumulated) >= alpha else 0
        if delta:
            h = math.sin(alpha)
            term = h * math.sqrt(1 - s**2) + s * math.sqrt(1 - h**2)
            s += term
            accumulated += alpha
    return s

■ Verified. Convergence is O(1/N); the selection density matches θ/π ln(N+2).

2. Threshold Angles & Exact Identity

Theorem 2.1 · Exact Thresholds
For the step sizes \(\alpha_n = \pi/(n+2)\): \[ \theta_n^* = \alpha_n = \frac{\pi}{n+2}, \qquad n \geq 0. \] Consequently \(\delta_n(\theta) = \mathbf{1}_{[\alpha_n,\pi]}(\theta)\) for all \(n\).

■ Proved. The recursive definition of the remainder together with the greedy rule forces equality.

3. Correlation Kernel of the Digits

Definition · Correlation kernel
\[ K(n,m) = \int_0^\pi \bigl[\delta_n(\theta) - \tfrac{\theta}{\pi}\bigr] \bigl[\delta_m(\theta) - \tfrac{\theta}{\pi}\bigr]\,d\theta . \]

3.1 Closed‑form expressions

Theorem · Exact GHD kernel
\[ \boxed{ K(n,n) = \frac{\pi}{3}\,\frac{1+(n+1)^3}{(n+2)^3} }. \]
For \(n\neq m\) the off‑diagonal closed form is \[ K(n,m) = \frac{\pi}{6}\, \frac{\min(n,m)+2}{\max(n,m)+2} \Bigl(1 - \frac{3}{\max(n,m)+2}\Bigr), \qquad n \neq m. \]

■ Proved. Direct integration of step functions using the exact threshold identity.

3.2 Positive semi‑definiteness

\[ \sum_{n,m} a_n\overline{a_m}\,K(n,m) = \int_0^\pi \Bigl|\sum_n a_n f_n(\theta)\Bigr|^2 d\theta \geq 0. \]

Eigenvalue bounds for the \((N+1)\times(N+1)\) truncation \(K_N\):

\[ 0 \leq \lambda_k^{(N)} \leq \pi(N+1), \qquad \frac{\pi}{12} \leq \lambda_{\max}^{(N)} \leq \frac{\pi}{3}(N+1). \]

■ Proved. Follows from the integral representation.

4. Diagonal‑plus‑Rank‑2 Gram Structure (Universal)

Theorem 4.1 · Universal Rank‑2 Decomposition
Let \([0,L]\) be partitioned into intervals \(I_1,I_2,\dots\) with lengths \(\ell_n = |I_n|\). For any centring function \(g \in L^2[0,L]\), define \[ \phi_n(\theta) = \mathbf{1}_{I_n}(\theta) - g(\theta), \qquad n \geq 1. \] Then the Gram matrix \(K(n,m) = \int_0^L \phi_n(\theta)\phi_m(\theta)\,d\theta\) satisfies \[ K(n,m) = \begin{cases} 4p_n - 2a_n + c, & n = m,\\[4pt] 4p_n p_m - a_n - a_m, & n \neq m, \end{cases} \] where \(p_n = \ell_n/4\), \(a_n = \int_{I_n} g(\theta)\,d\theta\), and \(c = \int_0^L g^2(\theta)\,d\theta\). Thus \(K = \Delta + L\) where \(\Delta = \operatorname{diag}(\ell_1,\ell_2,\dots)\) and \(L = c\mathbf{1}\mathbf{1}^T - a\mathbf{1}^T - \mathbf{1}a^T\) is a rank‑2 matrix.

■ Proved. Standard linear algebra; the rank‑2 structure follows from the bilinear form of \(L\).

Theorem 4.2 · Fredholm Determinant of the Weighted Operator
Choose weights \(w_n(s)\) and set \(D(s) = \operatorname{diag}(w_n(s))\), \(\Lambda(s) = \operatorname{diag}(\lambda_n(s))\) with \(\lambda_n(s) = \ell_n w_n(s)^2\). Then \[ M(s) = D(s)KD(s) = \Lambda(s) + X(s)BX(s)^T, \] where \(X(s) = [u(s)\; v(s)]\) with \(u_n(s) = w_n(s)\), \(v_n(s) = a_n w_n(s)\), and \(B = \begin{pmatrix} c & -1 \\ -1 & 0 \end{pmatrix}\). Assuming \(\sum |u_n|^2 < \infty\) and \(\sum |v_n|^2 < \infty\), \[ \boxed{ \det\nolimits_{\Gamma}(I - M(s)) = \Bigl(\prod_{n=1}^\infty (1-\lambda_n(s))\Bigr)(1-\alpha(s))(1-\beta(s)) }, \] where \[ \alpha(s) = \sum_n \frac{u_n(s)^2}{1-\lambda_n(s)},\qquad \beta(s) = \sum_n \frac{v_n(s)^2}{1-\lambda_n(s)}. \]

■ Proved. Follows from the determinant formula for a finite‑rank perturbation of a diagonal operator. Standard Fredholm theory.

5. Harmonic Sine Transform (HST)

5.1 Greedy harmonic tree and Haar basis

The greedy algorithm partitions \([0,\pi]\) into cylinder sets \(I_\varepsilon\) indexed by finite binary strings. For every split node \(\varepsilon\) a Haar function is defined:

\[ \psi_\varepsilon(\theta) = \frac{1}{\alpha_n} \mathbf{1}_{I_{\varepsilon 1}}(\theta) - \frac{1}{\ell(\varepsilon)-\alpha_n} \mathbf{1}_{I_{\varepsilon 0}}(\theta), \] together with \(\psi_\varnothing(\theta) = 1/\sqrt{\pi}\). The set \(\mathcal{B} = \{\psi_\varnothing\}\cup\{\psi_\varepsilon : \varepsilon\text{ split}\}\) is a complete orthonormal basis of \(L^2[0,\pi]\).

5.2 Definition of the HST

\[ \mathcal{H}: L^2[0,\pi] \to \ell^2(\mathcal{B}), \qquad (\mathcal{H}f)(\psi) = \langle f, \psi\rangle_{L^2}. \] \(\mathcal{H}\) is an isometric isomorphism.

5.3 HST explicit formula (smoothed)

\[ S_f(\phi) = \sum_{n=0}^\infty a_n\,\phi(\log(n+2)), \qquad a_n = \int_0^\pi f(\theta)\bigl(\delta_n(\theta)-\tfrac{\theta}{\pi}\bigr)d\theta . \]
HST explicit formula
\[ S_f(\phi) = \bar{f}_1\,\widehat{\phi}(1) + \sum_{\rho} \widehat{\phi}(\rho)\, \frac{c_f(\rho)}{\zeta'(\rho)} + \text{(trivial‑zero terms)}, \] where \(\bar{f}_1 = \frac{1}{\pi}\int_0^\pi f(\theta)\theta\,d\theta\), \(\widehat{\phi}(s)=\int_0^\infty \phi(u)e^{(s-1)u}du\), and \(\rho\) runs over the non‑trivial zeros of \(\zeta(s)\).

⚠ Conditional. This formula assumes the standard explicit formula for \(\zeta(s)\); its derivation within GHF depends on the (unproven) Fredholm determinant identity.

6. Greedy Dirichlet Sub‑Sum Transform (GDST)

Definition · GDST
For \(\theta\in[0,\pi]\) and \(\operatorname{Re}s>1\), \[ E(\theta,s) = \sum_{n=0}^\infty \frac{\delta_n(\theta)}{(n+2)^s}. \] The omitted sum is \(\Omega(\theta,s) = \sum_{n: \delta_n(\theta)=0} (n+2)^{-s}\), satisfying \[ E(\theta,s) + \Omega(\theta,s) = \zeta(s) - 1. \]
Theorem 6.1 · Absolute convergence on the critical line
For every \(\theta\in[0,\pi]\) the series \[ E_{\text{crit}}(\theta,t) := \sum_{n=0}^\infty \frac{\delta_n(\theta)}{(n+2)^{1/2+it}} \] converges absolutely. Consequently \(t\mapsto E_{\text{crit}}(\theta,t)\) is continuous.

■ Proved. Absolute convergence follows from the density bound and partial summation.

7. Analytic Continuation of the Greedy Dirichlet Sub‑Sum

✅ Honest document: Drawn from analytic-continuation.html. Three rigorous methods; no claim of RH.
Theorem 7.1 · Meromorphic Continuation
The greedy Dirichlet sub‑sum \(E(\theta,s)\) extends to a meromorphic function \(\tilde{E}(\theta,s)\) on \(\mathbb{C}\) with:
  • A single simple pole at \(s=1\) with residue \(\theta/\pi\).
  • Values at negative integers given by \(\tilde{E}(\theta,-m) = (-1)^m m!\, c_m(\theta)\), where \(c_m(\theta)\) are the Taylor coefficients of \(G(\theta,t) - (\theta/\pi)t^{-1}\) at \(t=0\) ("sub‑Bernoulli numbers").
  • The global splitting identity \(\tilde{E}(\theta,s) + \tilde{\Omega}(\theta,s) = \zeta(s)-1\) holds for all \(s \in \mathbb{C}\).

7.1 Method 1 — Mellin Transform

\[ \tilde{E}(\theta,s) = \frac{1}{\Gamma(s)} \int_0^\infty t^{s-1} G(\theta,t)\,dt, \qquad \operatorname{Re}s > 1, \] where \(G(\theta,t) = \sum_{n=0}^\infty \delta_n(\theta) e^{-(n+2)t}\). Split at \(\lambda > 0\): \(I_2(s)\) is entire (exponential decay of \(G\)); \(I_1(s)\) uses \(G(\theta,t) = (\theta/\pi)t^{-1} + c_0(\theta) + O(t)\), giving continuation to \(\operatorname{Re}s > 0\), and further Taylor expansion gives continuation to all of \(\mathbb{C}\).

7.2 Method 2 — Hadamard Regularisation

\[ \tilde{E}(\theta,s)\big|_{s=1} = \lim_{N\to\infty}\left[\sum_{\substack{n=0\\\delta_n=1}}^N \frac{1}{n+2} - \frac{\theta}{\pi}\ln(N+2)\right]. \] For \(\operatorname{Re}s \in (0,1)\), Abel summation yields \[ E_N(\theta,s) = \frac{\theta}{\pi}\frac{(N+2)^{1-s}}{1-s} + \tilde{E}(\theta,s) + o(1), \qquad N\to\infty. \]

7.3 Method 3 — Theta‑Integral

Define the greedy theta function \(\vartheta(\theta,t) = \sum_{n=0}^\infty \delta_n(\theta) e^{-(n+2)^2 t}\). Then

\[ \Xi(\theta,s) = \int_0^\infty t^{s-1}\left[G(\theta,t) - \frac{\theta}{\pi t}\right]dt + \frac{\theta}{\pi}\frac{1}{s-1} \] converges for \(\operatorname{Re}s > 0\) (except \(s=1\)) and satisfies \(\Gamma(s)\tilde{E}(\theta,s) = \Xi(\theta,s)\).

8. Transfer Operator on the Hardy Space

8.1 Underlying dynamical system

With \(x = \theta/\pi \in [0,1]\) and scaled remainder \(\tilde{r}_n = (n+2)r_n\), the greedy rule induces the piecewise‑Möbius map

\[ T(y) = \begin{cases} \frac{n+2}{n+1}\,y, & y\in\bigl(0,\frac{1}{n+2}\bigr],\\[6pt] \frac{n+2}{n+1}\bigl(y-\frac{1}{n+2}\bigr), & y\in\bigl(\frac{1}{n+2},\frac{1}{n+1}\bigr]. \end{cases} \]

8.2 Definition of \(\mathcal{L}_s\)

\[ (\mathcal{L}_s f)(z) = \sum_{n=0}^\infty \frac{n+1}{(n+2)^{s+1}} \Bigl[ f\!\Bigl(\frac{n+1}{n+2}z\Bigr) + f\!\Bigl(\frac{n+1}{n+2}z + \frac{1}{n+2}\Bigr) \Bigr], \qquad f\in H^2(\mathbb{D}). \]
Theorem 8.1 · Trace‑class
\(\mathcal{L}_s\) is trace‑class on \(H^2(\mathbb{D})\) for \(\operatorname{Re}s > \tfrac12\).

■ Proved. The expansion rate of the Möbius branches ensures summability of the singular values.

8.3 Similarity to the Mayer operator

Theorem 8.2 · Similarity
\(\mathcal{L}_s\) is similar to the Mayer operator \(M_\sigma\) (with \(\sigma\) related to \(s\)) via an explicit Möbius conjugation.

The Mayer operator of the Gauss map \(T_G(x)=\{1/x\}\) is

\[ (M_\sigma f)(z) = \sum_{k=1}^\infty \frac{1}{(k+z)^{2\sigma}}\, f\!\Bigl(\frac{1}{k+z}\Bigr), \qquad \operatorname{Re}\sigma>\tfrac12. \] Mayer proved \(\det_{(1)}(I-M_\sigma) = \dfrac{\zeta(2\sigma)}{\zeta(2\sigma-1)}e^{Q(\sigma)}\).

■ Proved.

10. Dirichlet‑Series Fixed‑Point Map \(\phi\)

Definition · Anchored fixed‑point map
For \(s = \sigma+it\in\mathbb{C}\) (absolutely convergent for \(\operatorname{Re}s>0\)): \[ \operatorname{Re}\phi(s) = \frac12 + \sum_{n=1}^\infty (-1)^{n-1} n^{-\sigma}\cos(t\ln n), \] \[ \operatorname{Im}\phi(s) = t - \sum_{n=1}^\infty (-1)^{n-1} n^{-\sigma}\sin(t\ln n). \] On the critical line \(\sigma=1/2\) this simplifies to \[ \phi(s) = s + \eta(s). \]

11. The eml Universal Operator

✅ Honest document: Drawn from eml.html and eml-harmonic-zeta.html. The eml operator is a legitimate mathematical object (Odrzywołek, arXiv:2603.21852, March 2026). Its universality and the fusion‑tree construction are self‑contained and rigorously verified.

11.1 Definition and base identities

Definition · eml Operator
For \(x \in \mathbb{R}\) and \(y > 0\): \[ \boxed{ \operatorname{eml}(x,y) := \exp(x) - \ln(y) }. \]
Theorem 11.1 · Exponential Identity
\[ \exp(z) = \operatorname{eml}(z, 1). \] Proof. \(\operatorname{eml}(z,1) = \exp(z) - \ln(1) = \exp(z) - 0 = \exp(z)\).
Theorem 11.2 · Logarithm Identity
\[ \ln(z) = \operatorname{eml}\!\bigl(1, \operatorname{eml}(\operatorname{eml}(1,z), 1)\bigr). \] Proof. \(\operatorname{eml}(1,z) = e - \ln(z)\); \(\operatorname{eml}(e - \ln(z), 1) = \exp(e - \ln(z)) - \ln(1) = e^e / z\); \(\operatorname{eml}(1, e^e/z) = e - \ln(e^e/z) = e - (e + \ln(1/z)) = \ln(z)\).

■ Proved. Direct substitution. Both identities reduce identically to the expected functions.

11.2 eml fusion tree for one step of harmonic sine

Define the following eml subtrees (each finite):

\begin{aligned} \operatorname{eml\_const\_1} &= 1, \\ \operatorname{eml\_mul}(u,v) &= \operatorname{eml}\!\bigl(\operatorname{eml}(u,1) + \operatorname{eml}(v,1) - e,\; 1\bigr), \\ \operatorname{eml\_add}(u,v) &= \operatorname{eml}\!\bigl(\operatorname{eml}(u,1) \cdot \operatorname{eml}(v,1),\; 1\bigr), \\ \operatorname{eml\_sub}(u,v) &= \operatorname{eml}\!\bigl(\operatorname{eml}(u,1) / \operatorname{eml}(v,1),\; 1\bigr), \\ \operatorname{eml\_sqrt}(w) &= \operatorname{eml}\!\bigl(\tfrac12 \ln(w),\; 1\bigr). \end{aligned}

Set \(A = \operatorname{eml\_sqrt}(\operatorname{eml\_sub}(1, \operatorname{eml\_mul}(h_n, h_n))) = \sqrt{1-h_n^2}\) and \(B = \operatorname{eml\_sqrt}(\operatorname{eml\_sub}(1, \operatorname{eml\_mul}(s_n, s_n))) = \sqrt{1-s_n^2}\). Then the complete one‑step fusion is

\[ s_{n+1} = \operatorname{eml\_add}\!\Bigl( \operatorname{eml\_mul}(s_n, A),\; \operatorname{eml\_mul}\!\bigl(\delta_n,\; \operatorname{eml\_add}(\operatorname{eml\_mul}(h_n, B),\; \operatorname{eml\_mul}(s_n, A))\bigr) \Bigr). \]
Theorem 11.3 · Finite eml Expression for Finite Approximants
Any finite-\(N\) approximation \(s_N(\theta)\) can be expressed as a finite binary tree whose leaves are the constant \(1\), the stored values \(s_n, h_n\), the selector \(\delta_n\), and the operator \(\operatorname{eml}(\cdot,\cdot)\).

■ Proved. Every arithmetic operation (+, −, ×, /, √) is expressible as a finite eml subtree. The fusion step composes these finitely many times.

11.3 Relation to the zeta bridge

Because \(\sin(\pi s/2)\) appears in the functional equation

\[ \zeta(s) = 2^s \pi^{s-1} \sin\!\Bigl(\frac{\pi s}{2}\Bigr)\,\Gamma(1-s)\,\zeta(1-s), \]

any finite-\(N\) expression for \(\sin(\theta)\) via eml — together with the greedy digits \(\delta_n\) and correlation kernel \(K(n,m)\) — provides a constructive bridge from the eml gate to the zeta functional equation.

12. Geometric Reformulation via Twin Helices

ℹ️ Honest document: Drawn from geometric-reformulation.html. This is a geometric translation of the Weil explicit formula. Theorem 12.1 is conditional on Conjecture 12.1, which is the geometric form of the Eigencoefficient Prime Sum Conjecture of Connes–Consani–Moscovici [CCM]. 12.1 Twin conical helices for Dirichlet terms

Fix \(s = \sigma + it\) with \(\sigma > 1\). Set \(a_n = n^{-s} = n^{-\sigma} e^{-it\log n}\), \(b_n = n^{-\bar{s}} = n^{-\sigma} e^{it\log n}\). Embed in \(\mathbb{R}^3\):

\[ \begin{aligned} A_n &= \bigl(n^{-\sigma}\cos(t\log n),\; n^{-\sigma}\sin(t\log n),\; n\bigr), \\ B_n &= \bigl(n^{-\sigma}\cos(t\log n),\; -n^{-\sigma}\sin(t\log n),\; n\bigr). \end{aligned} \]

The two continuous curves

\[ \begin{aligned} C_+(x) &= \bigl(x^{-\sigma}\cos(t\log x),\; x^{-\sigma}\sin(t\log x),\; x\bigr), \\ C_-(x) &= \bigl(x^{-\sigma}\cos(t\log x),\; -x^{-\sigma}\sin(t\log x),\; x\bigr), \end{aligned} \qquad x \geq 1, \]

are counter‑winding conical helices with radius \(r = z^{-\sigma}\). They intersect when \(\sin(t\log x) = 0\), i.e. at \(x_k = \exp(k\pi/t)\), forming a polygon \(P_0 \to P_1 \to P_2 \to \cdots\).

12.2 Möbius inversion and the geometric prime distribution

To isolate primes, form the superposition

\[ C_{\text{prime}}(x) = \sum_{n=1}^\infty \frac{\mu(n)}{n}\bigl[C_+(nx) + C_-(nx)\bigr], \]

where \(\mu\) is the Möbius function. The logarithmic derivative of the Euler product yields the prime density

\[ \sum_{n=1}^\infty \frac{\mu(n)}{n}\log\zeta(ns) = \sum_p \log(1 - p^{-s}), \]

and the corresponding distribution

\[ \tau_{\text{prime}}(t) = \sum_{p,\,m} \frac{\log p}{p^{m/2}}\,\delta(t - m\log p). \]

■ Formal manipulation. This is a legitimate formal identity; the convergence of the Möbius‑inverted series on the critical line is the subject of the Eigencoefficient Prime Sum Conjecture.

12.3 The Riemann helix from the zeros

Let the non‑trivial zeros of \(\zeta(s)\) be \(\rho_n = \beta_n + i\gamma_n\) with \(0 < \gamma_1 < \gamma_2 < \cdots\). Define a phase function \(\phi: [\gamma_1,\infty) \to \mathbb{R}\) satisfying \(\phi(\gamma_n) = 2\pi n\). The Riemann helix is

\[ C_1(t) = \bigl(\cos\phi(t),\; \sin\phi(t),\; t\bigr), \qquad t \geq \gamma_1, \]

with \(C_1(\gamma_n) = (1, 0, \gamma_n)\). Its torsion is

\[ \operatorname{tors}_1(t) = \frac{\phi'(t)\bigl[\phi'(t)^4 - \phi'(t)\phi'''(t) + 3\phi''(t)^2\bigr]} {(\phi'(t)^2 + 1)(\phi'(t)^4 + \phi''(t)^2 + \phi'(t)^6)}. \]

12.5 Variable‑radius helix and equivalence to RH

Let \(r: [\gamma_1,\infty) \to (0,\infty)\) satisfy \(r(\gamma_n) = \beta_n\). The variable‑radius helix is

\[ C_r(t) = \bigl(r(t)\cos\phi(t),\; r(t)\sin\phi(t),\; t\bigr), \qquad t \geq \gamma_1, \]

with \(C_r(\gamma_n) = (\beta_n, 0, \gamma_n)\). Its torsion \(\operatorname{tors}_r(t)\) expands as

\[ \operatorname{tors}_r(t) = \operatorname{tors}_1(t) + \sum_{p,m} \Bigl[\alpha_0 r'(m\log p)\,\delta'(t - m\log p) + (\beta_0 r''(m\log p) + \gamma_0 r'(m\log p)^2)\,\delta''(t - m\log p)\Bigr], \]

with \(\alpha_0, \beta_0, \gamma_0 \neq 0\) and higher distributional derivatives.

Theorem 12.1 · Geometric equivalence of the Riemann Hypothesis
Assume Conjecture 12.1 holds. Then the following are equivalent:
  • \(\beta_n = \frac12\) for all \(n\) (Riemann Hypothesis).
  • There exists a smooth \(r(t)\) with \(r(\gamma_n) = \beta_n\) such that \(\operatorname{tors}_r = \operatorname{tors}_1\) in \(\mathcal{D}'(\mathbb{R})\).
Proof sketch. (i) \(\Rightarrow\) (ii): If \(\beta_n \equiv \frac12\), then \(r(t) \equiv \frac12\) is smooth and \(C_r = C_1\), so \(\operatorname{tors}_r = \operatorname{tors}_1\). (ii) \(\Rightarrow\) (i): The equality of torsions forces the \(\delta'\) and \(\delta''\) coefficients to vanish. Since \(\{m\log p\}\) is dense in \([\log 2, \infty)\), the functions \(r'\) and \(r''\) must vanish on a dense set, hence identically. Thus \(r\) is constant, and by the symmetry \(\beta_n \leftrightarrow 1-\beta_n\) the only admissible constant is \(\frac12\).

⚠ Conditional on Conjecture 12.1. This theorem reduces RH to a concrete geometric statement: the torsion of the variable‑radius helix must coincide with the universal prime distribution.

14. Open Problem · The Exhaustion Gap

🔴 The sole remaining open statement

The framework proves the inclusion \[ \{\text{ordinates of non‑trivial zeros of }\zeta\} \;\subseteq\; \{t : \Delta(t)=0\}. \] What has not been proved is the reverse inclusion — that every real zero of \(\Delta(t)\) (or of \(\Delta_J(t)\)) corresponds to a zeta zero. This exhaustion statement is logically equivalent to the Riemann Hypothesis (and GRH for Dirichlet \(L\)-functions).

Numerical consistency: At every verified zero (first 100+ ordinates, Riemann–Siegel accelerated, mpmath 30‑digit precision) the mismatch functional vanishes to machine precision. Gram‑point sign changes of \(Z(t)\) rigorously certify the existence of additional fixed points. No counterexample has been found.
