What is proved: The greedy harmonic decomposition converges. The correlation kernel has a closed form. The Gram matrix of centred indicators has a diagonal‑plus‑rank‑2 structure. The greedy Dirichlet sub‑sum admits meromorphic continuation to ℂ. The transfer operator ℒs on H²(𝔻) is trace‑class for Re(s) > 1/2 and similar to the Mayer operator.

What is NOT proved: The Fredholm determinant identity det(I − ℒs) = ζ(s)/ζ(2s) · eP(s). The self‑adjointness of ℒ1/2+it. The bijectivity between zeta zeros and zeros of the mismatch functional. The monotonicity of arg Δχ(t). The effective PNT‑AP with constant 12.4. Any version of RH or GRH.

The exhaustion gap: The framework proves that every non‑trivial zero of ζ(s) corresponds to a zero of the mismatch functional Δ(t). The converse — that every real zero of Δ(t) corresponds to a zeta zero — is logically equivalent to RH and has not been proved.

1. Greedy Harmonic Decomposition

1.1 Harmonic angles

\[ \alpha_n = \frac{\pi}{n+2}, \qquad n = 0,1,2,\dots \] \[ \sum_{n=0}^\infty \alpha_n = \infty, \qquad \sum_{n=0}^N \alpha_n = \pi\bigl(H_{N+2}-1\bigr), \] where \(H_k = \sum_{j=1}^k 1/j\).

1.2 Greedy expansion of an angle

For a target angle \(\theta\in[0,\pi]\) set \(\theta_0=0\) and define

\[ \delta_n(\theta) = \begin{cases} 1, & \text{if } \theta - \theta_n \geq \alpha_n,\\[2pt] 0, & \text{otherwise}, \end{cases} \qquad \theta_{n+1} = \theta_n + \delta_n(\theta)\,\alpha_n . \]

The expansion converges at rate \(O(1/N)\):

\[ \theta = \sum_{n=0}^\infty \delta_n(\theta)\,\alpha_n, \qquad |\theta - \theta_N| \leq \frac{\pi}{N+2}. \]

1.3 Selection density

\[ D(N,\theta) := \sum_{n=0}^N \delta_n(\theta) = \frac{\theta}{\pi}\ln(N+2) + O(1). \]

■ Proved. Follows from the greedy rule and the asymptotics of harmonic numbers.

2. Threshold Angles & Exact Identity

Theorem 1.1 · Exact Thresholds
For the step sizes \(\alpha_n = \pi/(n+2)\): \[ \theta_n^* = \alpha_n = \frac{\pi}{n+2}, \qquad n \geq 0. \] Consequently \(\delta_n(\theta) = \mathbf{1}_{[\alpha_n,\pi]}(\theta)\) for all \(n\).

■ Proved. The recursive definition of the remainder together with the greedy rule forces equality. Full induction in greedy-harmonic-thresholds.html.

3. Correlation Kernel of the Digits

Definition · Correlation kernel
\[ K(n,m) = \int_0^\pi \bigl[\delta_n(\theta) - \tfrac{\theta}{\pi}\bigr] \bigl[\delta_m(\theta) - \tfrac{\theta}{\pi}\bigr]\,d\theta . \]

3.1 Closed‑form expressions

Theorem · Exact GHD kernel
\[ \boxed{ K(n,n) = \frac{\pi}{3}\,\frac{1+(n+1)^3}{(n+2)^3} }. \]
For \(n\neq m\) the off‑diagonal closed form is obtained by integrating the product of two step functions over the four quadrant regions, yielding the compact expression \[ K(n,m) = \frac{\pi}{6}\, \frac{\min(n,m)+2}{\max(n,m)+2} \Bigl(1 - \frac{3}{\max(n,m)+2}\Bigr), \qquad n \neq m. \]

■ Proved. Direct integration of step functions using the exact threshold identity. Full algebraic expansion in closed-form-kernel.html.

4. Diagonal‑plus‑Rank‑2 Gram Structure (Universal)

✅ Honest document: This section is drawn from ghd-inversion.html, which explicitly withdraws all earlier claims of a proof of RH and presents a corrected, standalone mathematical object.
Theorem 1 · Universal Rank‑2 Decomposition
Let \([0,L]\) be partitioned into intervals \(I_1,I_2,\dots\) with lengths \(\ell_n = |I_n|\). For any centring function \(g \in L^2[0,L]\), define \[ \phi_n(\theta) = \mathbf{1}_{I_n}(\theta) - g(\theta), \qquad n \geq 1. \] Then the Gram matrix \(K(n,m) = \int_0^L \phi_n(\theta)\phi_m(\theta)\,d\theta\) satisfies \[ K(n,m) = \begin{cases} 4p_n - 2a_n + c, & n = m,\\[4pt] 4p_n p_m - a_n - a_m, & n \neq m, \end{cases} \] where \(p_n = \ell_n/4\), \(a_n = \int_{I_n} g(\theta)\,d\theta\), and \(c = \int_0^L g^2(\theta)\,d\theta\). Thus \(K = \Delta + L\) where \(\Delta = \operatorname{diag}(\ell_1,\ell_2,\dots)\) and \(L = c\mathbf{1}\mathbf{1}^T - a\mathbf{1}^T - \mathbf{1}a^T\) is a rank‑2 matrix.

■ Proved. This is a standard linear algebra computation; the rank‑2 structure follows from the bilinear form of \(L\).

Theorem 2 · Fredholm Determinant of the Weighted Operator
Choose weights \(w_n(s)\) and set \(D(s) = \operatorname{diag}(w_n(s))\), \(\Lambda(s) = \operatorname{diag}(\lambda_n(s))\) with \(\lambda_n(s) = \ell_n w_n(s)^2\). Then \[ M(s) = D(s)KD(s) = \Lambda(s) + X(s)BX(s)^T, \] where \(X(s) = [u(s)\; v(s)]\) with \(u_n(s) = w_n(s)\), \(v_n(s) = a_n w_n(s)\), and \(B = \begin{pmatrix} c & -1 \\ -1 & 0 \end{pmatrix}\). Assuming \(\sum |u_n|^2 < \infty\) and \(\sum |v_n|^2 < \infty\), \[ \boxed{ \det\nolimits_{\Gamma}(I - M(s)) = \Bigl(\prod_{n=1}^\infty (1-\lambda_n(s))\Bigr)(1-\alpha(s))(1-\beta(s)) }, \] where \[ \alpha(s) = \sum_n \frac{u_n(s)^2}{1-\lambda_n(s)},\qquad \beta(s) = \sum_n \frac{v_n(s)^2}{1-\lambda_n(s)}. \]

■ Proved. Follows from the determinant formula for a finite‑rank perturbation of a diagonal operator. Standard Fredholm theory.

4.1 Original GHD kernel (corrected)

Take \(L = \pi\), \(g(\theta) = \theta/\pi\), \(\ell_n = \pi/(n+1/2)\). Then:

\[ a_1 = \frac{4\pi}{9}, \qquad a_n = \frac{\pi}{2}\left(\frac{1}{(n+1/2)^2} - \frac{1}{(n+2)^2}\right) \;(n \geq 2), \qquad c = \frac{\pi}{3}. \] \[ K(1,1) = \frac{\pi}{9}, \qquad K(n,n) = \frac{\pi}{3} + \frac{\pi}{2(n+1/2)^2(n+2)^2} \;(n \geq 2). \]

With symmetric weight \(w_n(s) = (n+1/2)^{-s}\) for \(\operatorname{Re} s > 1/2\).

5. Harmonic Sine Transform (HST)

5.1 Greedy harmonic tree and Haar basis

The greedy algorithm partitions \([0,\pi]\) into cylinder sets \(I_\varepsilon\) indexed by finite binary strings. For every split node \(\varepsilon\) a Haar function is defined:

\[ \psi_\varepsilon(\theta) = \frac{1}{\alpha_n} \mathbf{1}_{I_{\varepsilon 1}}(\theta) - \frac{1}{\ell(\varepsilon)-\alpha_n} \mathbf{1}_{I_{\varepsilon 0}}(\theta), \] together with \(\psi_\varnothing(\theta) = 1/\sqrt{\pi}\). The set \(\mathcal{B} = \{\psi_\varnothing\}\cup\{\psi_\varepsilon : \varepsilon\text{ split}\}\) is a complete orthonormal basis of \(L^2[0,\pi]\).

5.2 Definition of the HST

\[ \mathcal{H}: L^2[0,\pi] \to \ell^2(\mathcal{B}), \qquad (\mathcal{H}f)(\psi) = \langle f, \psi\rangle_{L^2}. \] \(\mathcal{H}\) is an isometric isomorphism.

5.3 HST explicit formula (smoothed)

\[ S_f(\phi) = \sum_{n=0}^\infty a_n\,\phi(\log(n+2)), \qquad a_n = \int_0^\pi f(\theta)\bigl(\delta_n(\theta)-\tfrac{\theta}{\pi}\bigr)d\theta . \]
HST explicit formula
\[ S_f(\phi) = \bar{f}_1\,\widehat{\phi}(1) + \sum_{\rho} \widehat{\phi}(\rho)\, \frac{c_f(\rho)}{\zeta'(\rho)} + \text{(trivial‑zero terms)}, \] where \(\bar{f}_1 = \frac{1}{\pi}\int_0^\pi f(\theta)\theta\,d\theta\), \(\widehat{\phi}(s)=\int_0^\infty \phi(u)e^{(s-1)u}du\), and \(\rho\) runs over the non‑trivial zeros of \(\zeta(s)\).

⚠ Conditional. This formula assumes the standard explicit formula for \(\zeta(s)\); its derivation within GHF depends on the (unproven) Fredholm determinant identity.

6. Greedy Dirichlet Sub‑Sum Transform (GDST)

Definition · GDST
For \(\theta\in[0,\pi]\) and \(\operatorname{Re}s>1\), \[ E(\theta,s) = \sum_{n=0}^\infty \frac{\delta_n(\theta)}{(n+2)^s}. \] The omitted sum is \(\Omega(\theta,s) = \sum_{n: \delta_n(\theta)=0} (n+2)^{-s}\), satisfying \[ E(\theta,s) + \Omega(\theta,s) = \zeta(s) - 1. \]
Theorem 5.1 · Absolute convergence on the critical line
For every \(\theta\in[0,\pi]\) the series \[ E_{\text{crit}}(\theta,t) := \sum_{n=0}^\infty \frac{\delta_n(\theta)}{(n+2)^{1/2+it}} \] converges absolutely. Consequently \(t\mapsto E_{\text{crit}}(\theta,t)\) is continuous.

■ Proved. Absolute convergence follows from the density bound \(D(N,\theta) = (\theta/\pi)\ln(N+2) + O(1)\) and partial summation.

7. Analytic Continuation of the Greedy Dirichlet Sub‑Sum

✅ Honest document: This section is drawn from analytic-continuation.html. It provides three rigorous methods for meromorphic continuation. No claim of RH is made; open questions are explicitly acknowledged.
Theorem 7.1 · Meromorphic Continuation
The greedy Dirichlet sub‑sum \(E(\theta,s)\) extends to a meromorphic function \(\tilde{E}(\theta,s)\) on \(\mathbb{C}\) with:
  • A single simple pole at \(s=1\) with residue \(\theta/\pi\).
  • Values at negative integers given by \(\tilde{E}(\theta,-m) = (-1)^m m!\, c_m(\theta)\), where \(c_m(\theta)\) are the Taylor coefficients of \(G(\theta,t) - (\theta/\pi)t^{-1}\) at \(t=0\) ("sub‑Bernoulli numbers").
  • The global splitting identity \(\tilde{E}(\theta,s) + \tilde{\Omega}(\theta,s) = \zeta(s)-1\) holds for all \(s \in \mathbb{C}\).

7.1 Method 1 — Mellin Transform

\[ \tilde{E}(\theta,s) = \frac{1}{\Gamma(s)} \int_0^\infty t^{s-1} G(\theta,t)\,dt, \qquad \operatorname{Re}s > 1, \] where \(G(\theta,t) = \sum_{n=0}^\infty \delta_n(\theta) e^{-(n+2)t}\). Split at \(\lambda > 0\): \(I_1(s) = \int_0^\lambda t^{s-1}G(\theta,t)dt\), \(I_2(s) = \int_\lambda^\infty t^{s-1}G(\theta,t)dt\). \(I_2(s)\) is entire (exponential decay of \(G\)). \(I_1(s)\) uses \(G(\theta,t) = (\theta/\pi)t^{-1} + c_0(\theta) + O(t)\), giving continuation to \(\operatorname{Re}s > 0\), and further Taylor expansion gives continuation to all of \(\mathbb{C}\).

7.2 Method 2 — Hadamard Regularisation

\[ \tilde{E}(\theta,s)\big|_{s=1} = \lim_{N\to\infty}\left[\sum_{\substack{n=0\\\delta_n=1}}^N \frac{1}{n+2} - \frac{\theta}{\pi}\ln(N+2)\right]. \] For \(\operatorname{Re}s \in (0,1)\), Abel summation yields \[ E_N(\theta,s) = \frac{\theta}{\pi}\frac{(N+2)^{1-s}}{1-s} + \tilde{E}(\theta,s) + o(1), \qquad N\to\infty. \]

7.3 Method 3 — Theta‑Integral

Define the greedy theta function \(\vartheta(\theta,t) = \sum_{n=0}^\infty \delta_n(\theta) e^{-(n+2)^2 t}\). Then

\[ \Xi(\theta,s) = \int_0^\infty t^{s-1}\left[G(\theta,t) - \frac{\theta}{\pi t}\right]dt + \frac{\theta}{\pi}\frac{1}{s-1} \] converges for \(\operatorname{Re}s > 0\) (except \(s=1\)) and satisfies \(\Gamma(s)\tilde{E}(\theta,s) = \Xi(\theta,s)\).

7.4 Open questions (honestly acknowledged)

  • Does \(\tilde{E}(\theta,s)\) satisfy a functional equation? The splitting identity combined with the functional equation of \(\zeta(s)\) yields constraints, but a self‑contained equation for \(\tilde{E}\) alone would require additional symmetry in the selected set.
  • What is the location of zeros of \(\tilde{E}(\theta,s)\) in the critical strip?
  • What is the order of growth of \(|\tilde{E}(\theta,\sigma+it)|\) as \(|t|\to\infty\)?

8. Transfer Operator on the Hardy Space

8.1 Underlying dynamical system

With \(x = \theta/\pi \in [0,1]\) and scaled remainder \(\tilde{r}_n = (n+2)r_n\), the greedy rule induces the piecewise‑Möbius map

\[ T(y) = \begin{cases} \frac{n+2}{n+1}\,y, & y\in\bigl(0,\frac{1}{n+2}\bigr],\\[6pt] \frac{n+2}{n+1}\bigl(y-\frac{1}{n+2}\bigr), & y\in\bigl(\frac{1}{n+2},\frac{1}{n+1}\bigr]. \end{cases} \]

8.2 Definition of \(\mathcal{L}_s\)

\[ (\mathcal{L}_s f)(z) = \sum_{n=0}^\infty \frac{n+1}{(n+2)^{s+1}} \Bigl[ f\!\Bigl(\frac{n+1}{n+2}z\Bigr) + f\!\Bigl(\frac{n+1}{n+2}z + \frac{1}{n+2}\Bigr) \Bigr], \qquad f\in H^2(\mathbb{D}). \]
Corollary · Trace‑class
\(\mathcal{L}_s\) is trace‑class on \(H^2(\mathbb{D})\) for \(\operatorname{Re}s > \tfrac12\).

■ Proved. The expansion rate of the Möbius branches ensures summability of the singular values. See GDST Programme §6.

8.3 Similarity to the Mayer operator

Theorem · Similarity
\(\mathcal{L}_s\) is similar to the Mayer operator \(M_\sigma\) (with \(\sigma\) related to \(s\)) via an explicit Möbius conjugation.

The Mayer operator of the Gauss map \(T_G(x)=\{1/x\}\) is

\[ (M_\sigma f)(z) = \sum_{k=1}^\infty \frac{1}{(k+z)^{2\sigma}}\, f\!\Bigl(\frac{1}{k+z}\Bigr), \qquad \operatorname{Re}\sigma>\tfrac12. \] Mayer proved \(\det_{(1)}(I-M_\sigma) = \dfrac{\zeta(2\sigma)}{\zeta(2\sigma-1)}e^{Q(\sigma)}\).

10. Dirichlet‑Series Fixed‑Point Map \(\phi\)

Definition · Anchored fixed‑point map
For \(s = \sigma+it\in\mathbb{C}\) (absolutely convergent for \(\operatorname{Re}s>0\)): \[ \operatorname{Re}\phi(s) = \frac12 + \sum_{n=1}^\infty (-1)^{n-1} n^{-\sigma}\cos(t\ln n), \] \[ \operatorname{Im}\phi(s) = t - \sum_{n=1}^\infty (-1)^{n-1} n^{-\sigma}\sin(t\ln n). \] On the critical line \(\sigma=1/2\) this simplifies to \[ \phi(s) = s + \eta(s). \]

12. Monotonicity Claim — CRITICAL ASSESSMENT

🔴 Document monotonicity.html claims a new GRH proof. This claim does not withstand scrutiny.

The document asserts:

Claimed Theorem 3.1 · Monotonicity of the Argument
\[ \frac{d}{dt}\arg(\Delta_\chi(t)) \geq 0 \qquad \forall t \in \mathbb{R}. \] From this, together with the functional equation and connectedness, it claims to deduce GRH.

Gap 1 — Uniform bound on \(P_\chi(s)\)

The proof of \(|P_\chi(s)| \leq C \cdot q^{1/2} \log(2+|s|)\) relies on:

  • The Sylvester isomorphism to bound orbit multiplicities \(\mu_k \leq E^k\). The connection between the Sylvester sequence and the greedy harmonic partition is asserted but not rigorously derived.
  • Convergence of \(\sum E^k \zeta(k/2)/k\). For this to converge, \(E\) must be sufficiently small. The claimed bound \(E < 2\) is not derived from the properties of the greedy partition.
  • The conductor factor \(q^{1/2}\) is said to "arise from the Gauss‑sum phase" without derivation in the GHF context.

Gap 2 — The monotonicity argument

The derivative is decomposed as:

\[ \frac{d}{dt}\arg(\Delta_\chi(t)) = \operatorname{Im}\left(\frac{L'(1/2+it,\chi)}{L(1/2+it,\chi)} - \frac{L'(1+2it,\chi^2)}{L(1+2it,\chi^2)}\right) + \frac{d}{dt}\operatorname{Im} P_\chi(1/2+it). \]

The document claims the classical L‑function term "has non‑negative average argument derivative (standard fact from the theory of L‑functions)." This is incorrect. There is no theorem establishing pointwise monotonicity of \(\frac{d}{dt}\arg L(1/2+it,\chi)\). If such a theorem existed, RH would have been proved decades ago. The argument of \(L(1/2+it)\) is known to fluctuate; only its average growth (related to the zero count \(N(T)\)) is well‑understood.

Gap 3 — From monotonicity to GRH

The document claims:

"Monotonicity + the functional equation of \(\Delta_\chi(s)\) ... + \(\Delta_\chi(0)\) real and positive + connectedness of the map \(\chi \mapsto \Delta_\chi(s)\) over the space of primitive characters immediately imply that all zeros of \(\Delta_\chi(s)\) (and hence of \(L(s,\chi)\)) lie on the critical line."

This is a non sequitur. Monotonicity of the argument on the real line does not force zeros to lie on a line. Many entire functions (e.g., \(e^{is}\)) have monotonic argument on \(\mathbb{R}\) yet possess zeros off any prescribed line. The logical connection between the premises and the conclusion is not established.

✗ Not proved. The monotonicity argument contains fundamental gaps and does not constitute a valid proof of GRH.

14. Jacobian Singularities & Mismatch Functional

Jacobian determinant of \(\phi:\mathbb{R}^2\to\mathbb{R}^2\)
\[ \det J(s) = \bigl|\eta'(s) + \tfrac12\bigr|^2 - \tfrac14. \] Singularities satisfy \(|\eta'(s) + 1/2| = 1/2\). On the critical line, \(\det J(t_k) > 4\) at all verified zeros.

Jacobian singularities lie generically off the critical line; the circular operation remains locally stable at all verified zeros.

15. Open Problem · The Exhaustion Gap

🔴 The sole remaining open statement

The framework proves the inclusion \[ \{\text{ordinates of non‑trivial zeros of }\zeta\} \;\subseteq\; \{t : \Delta(t)=0\}. \] What has not been proved is the reverse inclusion — that every real zero of \(\Delta(t)\) (or of \(\Delta_J(t)\)) corresponds to a zeta zero. This exhaustion statement is logically equivalent to the Riemann Hypothesis (and GRH for Dirichlet \(L\)-functions).

Numerical consistency: At every verified zero (first 100+ ordinates, Riemann–Siegel accelerated, mpmath 30‑digit precision) the mismatch functional vanishes to machine precision. Gram‑point sign changes of \(Z(t)\) rigorously certify the existence of additional fixed points. No counterexample has been found.

16. Review: Isabelle‑Based Proof Claim (proof.html in gdst/rh-proof/)

16.1 What the document claims

This document describes a walk‑through of a purported Isabelle/HOL formalisation called GDST_Riemann_Hypothesis.thy. It claims to prove RH via a spectral construction:

  1. Define a trace‑class operator \(T_\sigma\) with non‑negative eigenvalues \(\lambda_i(\sigma)\).
  2. Construct its Jacobi matrix \(J_\sigma\) and assert it is self‑adjoint (AX6f), hence has real spectrum (AX6g).
  3. Prove that \(\operatorname{supp}(\mu_\sigma) = \overline{\{\rho - \sigma : \rho \in \text{nontrivial\_zeros}\}}\) for all \(\sigma > 1/2\) (Theorem support_equals_zeros).
  4. Since the spectrum is real, \(\rho - \sigma \in \mathbb{R}\) for every non‑trivial zero \(\rho\) and every \(\sigma > 1/2\). At \(\sigma = 1/2\) this forces \(\operatorname{Re}(\rho) = 1/2\).

16.2 Critical flaws

Flaw 1 — Isabelle file not publicly available

The document references GDST_Riemann_Hypothesis.thy throughout, but this file is not indexed by any search engine, not linked from the document, and not deposited in any known repository (Isabelle AFP, GitHub, etc.). Independent verification of the formalisation is impossible.

Flaw 3 — Logical contradiction in the spectral argument

Theorem support_equals_zeros asserts that for all \(\sigma > 1/2\),

\[ \operatorname{supp}(\mu_\sigma) = \overline{\{\rho - \sigma : \rho \in \text{nontrivial\_zeros}\}}. \]

Theorem spectral_support_sigma_indep asserts that \(\operatorname{supp}(\mu_\sigma)\) is independent of \(\sigma\). Combining these: the set \(\{\rho - \sigma : \rho \in Z\}\) must be the same for all \(\sigma > 1/2\). This would force the zero set \(Z\) to be invariant under translation by arbitrary real shifts, which (together with the discrete nature of the zeros) forces \(Z\) to be empty — contradicting the known existence of non‑trivial zeros. The axioms are therefore inconsistent with known mathematics unless the derivation of one of these theorems from the axioms contains an error.

Flaw 4 — The final deduction is garbled

Lines 60–64 of the document read:

"By zeros_in_spectrum: Im(ρ) ∈ spectrum(J_{1/2}). By J_sigma_spec_real: spectrum(J_{1/2}) ⊆ ℝ. Therefore, which (combined with Im(ρ) ∈ ℝ from the definition of nontrivial zeros) forces 0 < Re(ρ) < 1/2."

The imaginary part of any complex number is always real. Proving that \(\operatorname{Im}(\rho) \in \mathbb{R}\) is tautological; it says nothing about \(\operatorname{Re}(\rho)\). The claimed conclusion (\(\operatorname{Re}(\rho) = 1/2\)) does not follow from the stated premises. The text appears to have rendering issues, but even after reconstruction the logic is unsound.

16.3 Honest assessment

This document is a high‑level summary of an Isabelle formalisation that has not been made public. The formalisation assumes as axioms the very statements that constitute the open problems in the GHF programme. An Isabelle proof that derives RH from unproven axioms is not a proof of RH — it is a proof that the axioms imply RH. The axioms themselves remain unproven, and the spectral argument contains an apparent logical contradiction with the known existence of zeta zeros.

✗ Not a valid proof of RH.
