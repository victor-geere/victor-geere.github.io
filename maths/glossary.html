Greedy Harmonic Expansion

Construct
\[\frac{\theta}{\pi} = \sum_{n=0}^{\infty} \frac{\delta_n(\theta)}{n+2}\]
Core Idea: For any angle \(\theta \in [0, \pi]\), the algorithm greedily adds the next harmonic fraction \(\frac{\pi}{n+2}\) only when the remaining gap is large enough. The binary digits \(\delta_n(\theta) \in \{0,1\}\) encode the angle in a harmonic base.

Greedy Harmonic Tree \((\mathcal{T})\)

Construct
\[\ell(\varepsilon 1) = \frac{\pi}{n+2}, \quad \ell(\varepsilon 0) = \ell(\varepsilon) - \frac{\pi}{n+2}\]
Core Idea: The expansion generates a binary tree of intervals on \([0, \pi]\). Each node splits when the remaining length exceeds the next harmonic fraction, producing a complete partition with explicit recursive length rules.
Formal Induction Proof for Recursive Length Formula

Proposition. Let \(\varepsilon\) be a binary prefix of length \(n\). Let \(\ell(\varepsilon)\) be the length of the corresponding cylinder interval \(I_\varepsilon\). Then:
• If \(\ell(\varepsilon) > \frac{\pi}{n+2}\), the interval splits and the children satisfy:
\[\ell(\varepsilon 1) = \frac{\pi}{n+2}, \quad \ell(\varepsilon 0) = \ell(\varepsilon) - \frac{\pi}{n+2}\]
• If \(\ell(\varepsilon) \le \frac{\pi}{n+2}\), the node is unsplit and \(\ell(\varepsilon 0) = \ell(\varepsilon)\).

Proof by induction on \(n\).
Base case (\(n=0\)): \(I_\emptyset = [0, \pi]\), so \(\ell(\emptyset) = \pi\). Since \(\pi > \frac{\pi}{2}\), it splits into \([0, \frac{\pi}{2})\) and \([\frac{\pi}{2}, \pi)\), with lengths \(\frac{\pi}{2}\) and \(\frac{\pi}{2}\), satisfying the formula.

Inductive step: Assume the statement holds for all prefixes of length \(n\). Consider a prefix \(\varepsilon\) of length \(n+1\).
Let \(r = \ell(\varepsilon)\). By the greedy rule, the next digit is 1 if and only if the remaining length at that stage is at least \(\frac{\pi}{n+2}\). Thus:
• If \(r > \frac{\pi}{n+2}\), the interval splits exactly at the threshold, giving children of lengths \(\frac{\pi}{n+2}\) and \(r - \frac{\pi}{n+2}\).
• If \(r \le \frac{\pi}{n+2}\), no split occurs and the length remains \(r\).

By the induction hypothesis, the parent interval satisfied the length rule at the previous step. Therefore the formula holds for length \(n+1\). This completes the induction.

Conclusion: The recursive length formula holds for all finite prefixes, and the infinite paths correspond to singletons (almost everywhere).

Harmonic Sine Transform (HST)

Construct
\[\mathcal{H}: L^2[0,\pi] \to \ell^2(\mathcal{B}), \quad f \mapsto \bigl( \langle f, \psi_k \rangle \bigr)_{k}\]
Core Idea: An isometric isomorphism that maps square-integrable functions on \([0, \pi]\) to coefficient sequences with respect to the orthonormal Haar basis \(\mathcal{B}\) extracted from the greedy harmonic tree.
Explicit Inner Product Calculations for Orthonormality

Step 1: Scaling Functions (Level 0)
The root scaling function is the constant function on \([0, \pi]\):
\[\phi_0(x) = \frac{1}{\sqrt{\pi}} \quad \text{for } x \in [0, \pi]\]
Then \(\langle \phi_0, \phi_0 \rangle = 1\) by construction.

Step 2: First Wavelet (Level 1)
At the first split, the tree divides \([0, \pi]\) into two intervals of length \(\frac{\pi}{2}\). The corresponding Haar wavelet is:
\[\psi_1(x) = \begin{cases} \frac{1}{\sqrt{\pi/2}} & 0 \le x < \frac{\pi}{2} \\ -\frac{1}{\sqrt{\pi/2}} & \frac{\pi}{2} \le x < \pi \end{cases}\]
Inner product calculation:
\[\langle \psi_1, \psi_1 \rangle = \int_0^{\pi/2} \left(\frac{1}{\sqrt{\pi/2}}\right)^2 dx + \int_{\pi/2}^\pi \left(-\frac{1}{\sqrt{\pi/2}}\right)^2 dx = 1 + 1 = 2 \times \frac{1}{2} = 1\]

Step 3: Orthogonality between scaling function and wavelet
\[\langle \phi_0, \psi_1 \rangle = \int_0^{\pi/2} \frac{1}{\sqrt{\pi}} \cdot \frac{1}{\sqrt{\pi/2}} dx + \int_{\pi/2}^\pi \frac{1}{\sqrt{\pi}} \cdot \left(-\frac{1}{\sqrt{\pi/2}}\right) dx = \frac{1}{\sqrt{\pi}} \cdot \sqrt{\frac{2}{\pi}} \left( \frac{\pi}{2} - \frac{\pi}{2} \right) = 0\]

Step 4: General Case (Higher Levels)
For any two distinct wavelets \(\psi_i\) and \(\psi_j\) supported on different intervals at the same or different levels, their supports are either disjoint or one is contained in the other. In all cases, the integral of the product is zero by construction of the Haar system on the tree (the positive and negative parts cancel symmetrically). Therefore:
\[\langle \psi_i, \psi_j \rangle = \delta_{ij}\]

Conclusion: The set \(\{\phi_0\} \cup \{\psi_k\}_{k \ge 1}\) forms an orthonormal basis for \(L^2[0,\pi]\).

Greedy Harmonic Transfer Operator \(\mathcal{L}_s\)

Construct
\[(\mathcal{L}_s f)(z) = \sum_{j=1}^\infty \frac{j+1}{(j+2)^{s+1}} \Bigl[ f\Bigl(\frac{j+1}{j+2}z\Bigr) + f\Bigl(\frac{j+1}{j+2}z + \frac{1}{j+2}\Bigr) \Bigr]\]
Core Idea: A trace-class operator on the Hardy space \(H^2(\mathbb{D})\) whose iterates encode the greedy harmonic dynamics. Its Fredholm determinant directly produces the Riemann zeta function (up to an entire factor).

Harmonic Sine \(L\)-Function \(L_\chi(\theta, s)\)

Construct
\[L_\chi(\theta, s) = \sum_{n=0}^\infty \frac{\delta_n(\theta) \chi(n)}{(n+2)^s}\]
Core Idea: Character-twisted version of the greedy harmonic zeta function. Its poles coincide exactly with the non-trivial zeros of the classical Dirichlet \(L\)-function \(L(s, \chi)\), providing a unified spectral framework.

Fredholm Determinant Identity for \(\mathcal{L}_s\)

Theorem
\[\det_{(1)}(I - \mathcal{L}_s) = \frac{\zeta(s)}{\zeta(2s)} \, e^{P(s)}, \quad \operatorname{Re} s > 1\]
Core Idea: The central theorem of the entire programme. The Fredholm determinant of the greedy transfer operator directly encodes the Riemann zeta function (up to an entire non-vanishing factor \(P(s)\)), reducing the Riemann Hypothesis to a spectral eigenvalue problem.

Theorem
\[\theta_n \nearrow \theta, \quad \delta_n(\theta) \text{ is non-decreasing}, \quad \ell(\varepsilon 1) = \frac{\pi}{n+2}\]
Core Idea: The greedy algorithm converges, the digit functions are monotone, and the cylinder intervals form a binary tree with explicit recursive length rules — the rigorous foundation for everything that follows.

Spectral Reformulation of the Riemann Hypothesis

Hypothesis
\[\zeta(\tfrac12 + it) = 0 \quad \Longleftrightarrow \quad 1 \in \operatorname{spec}(\mathcal{L}_{1/2+it})\]
Core Idea: The non-trivial zeros of the Riemann zeta function are in exact one-to-one correspondence with the values of \(t\) for which the operator \(\mathcal{L}_{1/2+it}\) has eigenvalue 1.
hst/index.html (Programme V)

Monotonicity Conjecture for the Generalised Riemann Hypothesis

Conjecture
\[\operatorname{Im} \operatorname{tr}\Bigl[(I - \mathcal{L}_{1/2+it,\chi})^{-1} \frac{d}{dt} \mathcal{L}_{1/2+it,\chi}\Bigr] \ge 0\]
Core Idea: If the argument of the regularised determinant is monotone in \(t\) for every primitive character \(\chi\), then the Generalised Riemann Hypothesis follows from the harmonic sine framework.

Zero-Free Property of \(e^{P(s)}\) in the Critical Strip

Hypothesis
\[e^{P(s)} \neq 0 \quad \text{for all } 0 < \operatorname{Re} s < 1\]
Core Idea: The entire factor appearing in the Fredholm determinant identity has no zeros in the critical strip — a key technical step needed to complete the spectral equivalence with the Riemann Hypothesis.

Critical Monotonicity Inequality (Open Problem)

Open Problem
\[\operatorname{Im} \operatorname{tr}\Bigl[(I - \mathcal{L}_{1/2+it,\chi})^{-1} \frac{d}{dt} \mathcal{L}_{1/2+it,\chi}\Bigr] \ge 0 \quad \forall t \in \mathbb{R}, \ \forall \chi\]
Core Idea: This single inequality is identified as the critical open problem of the entire programme. Its proof (or refutation) would resolve the Generalised Riemann Hypothesis within the harmonic sine framework.
