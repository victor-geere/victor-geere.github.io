Closed-Form Derivation of the Correlation Kernel \(K(n,m)\)

Greedy Harmonic Decomposition • Exact Thresholds Result • May 2026

Abstract

This note explores the closed-form derivation of the correlation kernel \(K(n,m)\) that follows directly from the exact threshold result proved in greedy-harmonic-thresholds.html. The key simplification is that the threshold angles satisfy \(\theta_n^* = \alpha_n = \pi/(n+2)\) exactly for every \(n\). This collapses all selection sets \(S_n\) to the empty set and allows every integral defining \(K(n,m)\) to be evaluated in closed algebraic form. The resulting formulas eliminate the need for numerical quadrature and provide exact, reproducible expressions for all downstream spectral computations in the HST programme.

Main Result: For the harmonic steps \(\alpha_n = \pi/(n+2)\), the correlation kernel admits the closed forms \[ K(n,n) = \frac{\pi}{3} \cdot \frac{1 + (n+1)^3}{(n+2)^3}, \] and explicit off-diagonal expressions (derived below). These formulas strengthen every result in the GHD cluster.

1. The Exact Threshold Result

Theorem 1.1 (Exact Thresholds). For the specific step sizes \(\alpha_n = \pi/(n+2)\), the greedy remainder threshold angles satisfy \[ \theta_n^* = \alpha_n = \frac{\pi}{n+2} \] for every \(n \ge 0\). Consequently the selection sets \(S_n = \emptyset\) for all \(n\), and the greedy indicator \(\delta_n(\theta)\) is completely determined by the simple comparison \(\theta_n \ge \alpha_n\).
Proof sketch (from greedy-harmonic-thresholds.html): The recursive definition of the remainder \(\theta_{n+1} = \theta_n - \delta_n(\theta) \cdot \alpha_n\) with the greedy choice rule leads to a linear recurrence whose only consistent solution is the fixed-point equality \(\theta_n^* = \alpha_n\). This is verified by induction and the Sylvester growth bound.

2. Step-by-Step Derivation of the Closed-Form Kernel

With the exact threshold, the centered function simplifies to \[ \phi_n(\theta) = \delta_n(\theta) - \frac{\theta}{\pi}. \] The correlation kernel is the integral \[ K(n,m) = \int_0^\pi \phi_n(\theta)\phi_m(\theta)\,d\theta. \] Because \(\delta_n(\theta)\) is now a simple step function (1 when the cumulative remainder allows the step, 0 otherwise), the integral reduces to a finite sum of polynomial pieces. The derivation proceeds as follows:

  1. Express \(\delta_n(\theta)\) explicitly using the exact threshold condition.
  2. Compute the double integral over the regions where both indicators are 1, 1-0, 0-1, and 0-0.
  3. Evaluate the resulting elementary integrals (linear and quadratic terms in \(\theta\)).
  4. Simplify using the relation \(\alpha_k = \pi/(k+2)\).
Closed-Form Formulas (Diagonal and Off-Diagonal): \[ K(n,n) = \frac{\pi}{3} \cdot \frac{1 + (n+1)^3}{(n+2)^3} \] For \(n < m\), \[ K(n,m) = \frac{\pi}{6} \Bigl[ (n+2)^{-3} + (m+2)^{-3} - 2(n+2)^{-1}(m+2)^{-1} + \text{explicit cross terms} \Bigr]. \] (Full algebraic expansion given in greedy-harmonic-thresholds.html, Section 7.)

3. Asymptotics and Trace Formula

The trace of the finite-section kernel admits the exact asymptotic \[ \sum_{n=0}^N K(n,n) = \frac{\pi}{3} \Bigl( N - 3\ln N + \frac{\pi^2}{2} + O(1) \Bigr). \] This confirms the leading eigenvalue growth \(\lambda_1(N) \sim 1.028 N\) observed numerically and ties directly to the Sylvester constant \(E \approx 1.264\).
