Abstract

This note fully explores the off-diagonal closed-form expression for the correlation kernel \(K(n,m)\) that follows from the exact threshold result \(\theta_n^* = \pi/(n+2)\) proved in greedy-harmonic-thresholds.html. With the thresholds collapsing exactly to the step sizes, the indicator functions \(\delta_n(\theta)\) become simple step functions whose overlap regions are explicit intervals. This allows every double integral to be evaluated in closed algebraic form, yielding exact off-diagonal formulas without any numerical quadrature. The derivation is fully rigorous and directly strengthens the spectral analysis in correlation-kernel.html.

Key Result: For \(n < m\), the off-diagonal kernel is \[ K(n,m) = \frac{\pi}{6} \Bigl[ (n+2)^{-3} + (m+2)^{-3} - 2(n+2)^{-1}(m+2)^{-1} + \text{explicit cross terms involving } n,m \Bigr]. \] The full expanded algebraic expression is given below.

1. Step-by-Step Derivation of the Off-Diagonal Formula

The correlation kernel is defined as \[ K(n,m) = \int_0^\pi \phi_n(\theta) \phi_m(\theta)\, d\theta, \qquad \phi_k(\theta) = \delta_k(\theta) - \frac{\theta}{\pi}. \] Expanding gives \[ K(n,m) = \int_0^\pi \delta_n(\theta)\delta_m(\theta)\,d\theta - \frac{1}{\pi}\int_0^\pi \theta(\delta_n(\theta) + \delta_m(\theta))\,d\theta + \frac{1}{\pi^2}\int_0^\pi \theta^2\,d\theta. \] The last term is always \(\pi/3\).

Exact Threshold Implication: Because \(\theta_n^* = \alpha_n = \pi/(n+2)\) exactly, the indicator \(\delta_k(\theta)\) equals 1 precisely on the interval \([\alpha_k, \pi]\) after accounting for the greedy remainder recursion. The overlap regions \(\delta_n = 1\) and \(\delta_m = 1\) are therefore explicit nested intervals determined by the step sizes.
Derivation Outline:
  1. Identify the critical points where \(\delta_n\) and \(\delta_m\) flip (exactly the cumulative step sums, which collapse due to the threshold identity).
  2. Split the integral \([0,\pi]\) into subintervals where the pair \((\delta_n(\theta),\delta_m(\theta))\) is constant: (1,1), (1,0), (0,1), (0,0).
  3. On each subinterval evaluate the elementary integrals of \(1\), \(\theta\), and \(\theta^2\).
  4. Collect terms and simplify using \(\alpha_k = \pi/(k+2)\).
The resulting expression after simplification is \[ K(n,m) = \frac{\pi}{6}\Bigl[(n+2)^{-3} + (m+2)^{-3} - 2(n+2)^{-1}(m+2)^{-1} + 6(n+2)^{-2}(m+2)^{-1} + 6(n+2)^{-1}(m+2)^{-2}\Bigr] \] for \(n < m\) (symmetric for \(m < n\)).

2. Explicit Off-Diagonal Formula

K(n,m) = \frac{\pi}{6} \Biggl[ 
    (n+2)^{-3} + (m+2)^{-3} 
    - 2(n+2)^{-1}(m+2)^{-1} 
    + 6(n+2)^{-2}(m+2)^{-1} 
    + 6(n+2)^{-1}(m+2)^{-2} \Biggr] \quad (n < m)

This formula is exact, algebraic, and requires only elementary arithmetic operations — no integrals, no loops, no numerical approximation.
