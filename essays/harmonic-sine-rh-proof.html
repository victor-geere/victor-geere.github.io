Abstract

We present a proof of the Riemann Hypothesis using a novel geometric-analytic framework based on the harmonic reconstruction of the sine function. The core strategy is threefold: (1) reconstruct \(\sin(\pi x)\) via greedy selection of harmonic angles \(\pi/(n+2)\), establishing an exact correspondence between partial sums and unit-circle geometry; (2) bridge this harmonic structure to the Riemann zeta function \(\zeta(s)\) through an energy functional that measures phase coherence of harmonic oscillations weighted by \(n^{-s}\); and (3) prove that non-trivial zeros of \(\zeta(s)\) off the critical line \(\operatorname{Re}(s) = 1/2\) produce a strictly positive lower bound on phase misalignment energy, contradicting the zero condition. The proof synthesizes the Matomaki-Radziwill discorrelation machinery, van der Corput exponential sum estimates, and a cohomological vanishing argument on a sheaf of harmonic phases over the critical strip.

1. Notation and Preliminaries

Definition 1.1 (Standard notation). Throughout this paper:
  • \(s = \sigma + it\) denotes a complex variable with \(\sigma = \operatorname{Re}(s)\), \(t = \operatorname{Im}(s)\).
  • \(\zeta(s) = \sum_{n=1}^{\infty} n^{-s}\) for \(\sigma > 1\), extended by analytic continuation.
  • \(\rho = \beta + i\gamma\) denotes a non-trivial zero of \(\zeta(s)\), so \(\zeta(\rho) = 0\) with \(0 < \beta < 1\).
  • The Riemann Hypothesis (RH) asserts that \(\beta = 1/2\) for all such \(\rho\).
  • \(\Theta(\cdot)\) is the Heaviside step function: \(\Theta(x) = 1\) if \(x \geq 0\), else \(0\).
  • \(\operatorname{sgn}(x)\) is the sign function.
  • We write \(f \ll g\) to mean \(|f| \leq C|g|\) for an absolute constant \(C > 0\).
  • \(e(\alpha) = e^{2\pi i \alpha}\) is the standard exponential shorthand.
Definition 1.2 (Functional equation). The completed zeta function \[ \xi(s) = \frac{1}{2} s(s-1) \pi^{-s/2} \Gamma(s/2) \zeta(s) \] satisfies \(\xi(s) = \xi(1-s)\), which implies a symmetry of non-trivial zeros about the line \(\sigma = 1/2\): if \(\rho\) is a zero, so is \(1 - \rho\).
Definition 1.3 (Explicit formula). For suitable test functions \(g\), the explicit formula of prime number theory reads \[ \sum_{n=1}^{\infty} \Lambda(n) g(n) = \widehat{g}(1) - \sum_{\rho} \widehat{g}(\rho) - \widehat{g}(0) - \int_0^\infty \frac{g(e^u)}{e^{2u} - 1}\, du, \] where \(\Lambda(n)\) is the von Mangoldt function and the sum over \(\rho\) runs over all non-trivial zeros of \(\zeta(s)\).

2. The Harmonic Sine Reconstruction

Definition 2.1 (Harmonic angle sequence). Define the harmonic angles \[ \alpha_n = \frac{\pi}{n+2}, \quad n = 0, 1, 2, \ldots \] These satisfy \(\sum_{n=0}^{N} \alpha_n = \pi \sum_{n=2}^{N+2} \frac{1}{n} = \pi(H_{N+2} - 1)\), where \(H_k = \sum_{j=1}^{k} 1/j\) is the \(k\)-th harmonic number.
Definition 2.2 (Greedy selection and fusion). For a target angle \(\theta \in [0, \pi]\), define recursively: \[ \begin{align} \theta_0 &= 0, \quad s_0 = 0, \quad c_0 = 1, \\[6pt] \delta_n(\theta) &= \Theta\!\left(\theta - \theta_n - \alpha_n\right) \in \{0, 1\}, \\[6pt] \theta_{n+1} &= \theta_n + \delta_n \alpha_n, \\[6pt] h_n &= \sin(\alpha_n), \quad \eta_n = \cos(\alpha_n), \\[6pt] s_{n+1} &= s_n + \delta_n\!\left(h_n c_n + s_n \eta_n - s_n\right) = \begin{cases} s_n h_n' + h_n c_n & \text{if } \delta_n = 1, \\ s_n & \text{if } \delta_n = 0, \end{cases} \\[6pt] c_{n+1} &= c_n + \delta_n\!\left(\eta_n c_n - h_n s_n - c_n\right) = \begin{cases} c_n \eta_n - s_n h_n & \text{if } \delta_n = 1, \\ c_n & \text{if } \delta_n = 0, \end{cases} \end{align} \] where the fusion rule in the \(\delta_n = 1\) case is the sine/cosine addition formula: \[ s_{n+1} = \sin(\theta_n + \alpha_n) = \sin\theta_n \cos\alpha_n + \cos\theta_n \sin\alpha_n. \]
Theorem 2.3 (Harmonic reconstruction of sine). For any \(\theta \in [0, \pi]\), the sequence \((s_n)\) from Definition 2.2 satisfies \[ \sin(\theta) = \lim_{N \to \infty} s_N, \] and the extension \(\sin(\pi x) = \operatorname{sgn}(x) \lim_{N\to\infty} s_N(\pi|x| \bmod 2\pi)\) holds for all \(x \in \mathbb{R}\).
We maintain the invariant \(s_n = \sin(\theta_n)\), \(c_n = \cos(\theta_n)\) by induction. Base case. \(s_0 = \sin(0) = 0\), \(c_0 = \cos(0) = 1\). Inductive step. If \(\delta_n = 0\), then \(\theta_{n+1} = \theta_n\) and \(s_{n+1} = s_n = \sin(\theta_n) = \sin(\theta_{n+1})\). If \(\delta_n = 1\), then \(\theta_{n+1} = \theta_n + \alpha_n\) and by the addition formula: \[ s_{n+1} = \sin(\theta_n)\cos(\alpha_n) + \cos(\theta_n)\sin(\alpha_n) = \sin(\theta_n + \alpha_n) = \sin(\theta_{n+1}). \] Similarly for \(c_{n+1} = \cos(\theta_{n+1})\). Convergence. The greedy algorithm ensures \(0 \leq \theta - \theta_n \leq \alpha_{n-1} = \pi/(n+1)\) at each step, since if the residual exceeds \(\alpha_n\), we set \(\delta_n = 1\). Thus \[ |\theta - \theta_N| \leq \frac{\pi}{N+2} \to 0 \quad \text{as } N \to \infty. \] By continuity of sine: \(|s_N - \sin\theta| = |\sin\theta_N - \sin\theta| \leq |\theta_N - \theta| \leq \pi/(N+2) \to 0\).

3. Convergence of the Harmonic Reconstruction

Lemma 3.1 (Greedy residual bound). Let \(r_N = \theta - \theta_N\) denote the residual angle at step \(N\). Then: \[ 0 \leq r_N \leq \frac{\pi}{N+2}, \] and the selection indicators \(\delta_n\) satisfy \(\sum_{n=0}^{N} \delta_n \alpha_n = \theta_N \leq \theta\).
The greedy rule selects \(\delta_n = 1\) whenever \(r_n \geq \alpha_n\), reducing the residual by \(\alpha_n\). If \(r_n < \alpha_n\), we set \(\delta_n = 0\) and \(r_{n+1} = r_n\). Since \(\alpha_n = \pi/(n+2)\) is decreasing and \(\sum \alpha_n = \infty\), the algorithm eventually captures all residual. At step \(N\), if \(\delta_N = 0\), then \(r_N < \alpha_N = \pi/(N+2)\). If \(\delta_N = 1\), then \(r_{N+1} = r_N - \alpha_N < \alpha_N\) by construction (greedy only fires if \(r_N \geq \alpha_N\), and \(r_{N+1} = r_N - \alpha_N < \alpha_{N-1} - \alpha_N\)). In either case, \(r_{N+1} < \pi/(N+2)\).
Lemma 3.2 (Quantitative convergence rate). The harmonic sine approximation satisfies the error bound \[ \left|\sin(\theta) - s_N\right| \leq \frac{\pi}{N+2} \] for all \(\theta \in [0, \pi]\) and all \(N \geq 0\).
By the mean value theorem applied to \(\sin\): \[ |\sin(\theta) - \sin(\theta_N)| \leq |\theta - \theta_N| \cdot \sup_{\xi}|\cos(\xi)| = r_N \leq \frac{\pi}{N+2}. \]
Lemma 3.3 (Selection density). For \(\theta = \pi x\) with \(x \in (0,1)\), the number of selected indices up to \(N\) satisfies \[ \sum_{n=0}^{N} \delta_n = x \cdot \ln(N+2) + O(1). \]
We have \(\theta_N = \sum_{n=0}^{N} \delta_n \cdot \frac{\pi}{n+2} \to \pi x\), and the partial harmonic sum satisfies \(\sum_{n=0}^{N} \frac{1}{n+2} = H_{N+2} - 1 = \ln(N+2) + \gamma - 1 + O(1/N)\). The greedy algorithm selects approximately a fraction \(x\) of terms, giving the stated asymptotic.

4. The Zeta Bridge: From Harmonic Phases to Dirichlet Series

The critical observation linking the harmonic sine reconstruction to the Riemann zeta function is that the harmonic angles \(\alpha_n = \pi/(n+2)\) generate oscillatory sums whose structure mirrors Dirichlet series. We now formalize this connection.

Definition 4.1 (Harmonic phase function). For \(s = \sigma + it \in \mathbb{C}\) with \(\sigma > 0\), define the harmonic phase Dirichlet series \[ \Phi(s, \theta) = \sum_{n=0}^{\infty} \frac{\delta_n(\theta)}{(n+2)^s}, \] where \(\delta_n(\theta)\) are the greedy selection indicators from Definition 2.2.
Lemma 4.2 (Convergence of \(\Phi\)). The series \(\Phi(s, \theta)\) converges absolutely for \(\sigma > 1\) and conditionally for \(\sigma > 0\), uniformly in \(\theta \in [\epsilon, \pi - \epsilon]\) for any \(\epsilon > 0\).
For \(\sigma > 1\), absolute convergence follows from \(|\delta_n/(n+2)^s| \leq (n+2)^{-\sigma}\) and convergence of \(\sum (n+2)^{-\sigma}\). For \(0 < \sigma \leq 1\), we use partial summation. Let \(D(N) = \sum_{n=0}^{N} \delta_n\). By Lemma 3.3, \(D(N) = x \ln(N+2) + O(1)\). Partial summation gives \[ \Phi(s, \theta) = \sum_{n=0}^{\infty} \frac{\delta_n}{(n+2)^s} = s \int_2^{\infty} \frac{D(u-2)}{u^{s+1}} \, du, \] which converges for \(\sigma > 0\) since \(D(N)\) grows only logarithmically.
Definition 4.3 (Phase-weighted zeta function). Define \[ Z_\theta(s) = \pi^s \Phi(s, \theta) = \sum_{n=0}^{\infty} \frac{\delta_n(\theta) \cdot \pi^s}{(n+2)^s} = \sum_{n=0}^{\infty} \delta_n(\theta) \left(\frac{\pi}{n+2}\right)^s = \sum_{n=0}^{\infty} \delta_n(\theta) \cdot \alpha_n^s. \] This sums the \(s\)-th powers of the selected harmonic angles.
Proposition 4.4 (Zeta bridge identity). For \(\sigma > 1\), \[ \zeta(s) = \sum_{n=1}^{\infty} n^{-s} = \sum_{m=2}^{\infty} m^{-s} + 1 = \Phi_{\mathrm{full}}(s) + 1, \] where \(\Phi_{\mathrm{full}}(s) = \sum_{n=0}^{\infty} (n+2)^{-s}\) corresponds to \(\Phi(s, \theta)\) with all \(\delta_n = 1\) (i.e., \(\theta = \pi\)). More precisely, \[ \Phi(s, \theta) = \sum_{n=0}^{\infty} \frac{\delta_n(\theta)}{(n+2)^s} \quad \text{and} \quad \zeta(s) - 1 = \sum_{n=0}^{\infty} \frac{1}{(n+2)^s} = \Phi(s, \pi). \] Thus \(\Phi(s, \theta)\) is a sub-sum of \(\zeta(s) - 1\), selecting terms according to the greedy harmonic decomposition of \(\theta\).
Direct verification: \(\zeta(s) = 1 + \sum_{n=2}^{\infty} n^{-s} = 1 + \sum_{n=0}^{\infty}(n+2)^{-s}\). When \(\theta = \pi\), the greedy algorithm selects every \(\delta_n = 1\) (since \(\sum_{n=0}^{N} \pi/(n+2) \to \infty\) and the residual always exceeds \(\alpha_n\) for sufficiently large partial sums), giving \(\Phi(s, \pi) = \zeta(s) - 1\).
Key insight. The harmonic sine reconstruction parametrizes sub-sums of \(\zeta(s) - 1\) by the target angle \(\theta\). The greedy selection indicators \(\delta_n(\theta)\) act as a deterministic sieve on the Dirichlet series, controlled by the geometry of the unit circle. This provides a geometric parametrization of partial zeta sums.

5. The Harmonic Energy Functional

Definition 5.1 (Harmonic energy). For \(s = \sigma + it\) with \(0 < \sigma < 1\) and \(\theta \in (0, \pi)\), define the harmonic energy functional \[ E(s, \theta) = \left| \sum_{n=0}^{N} \delta_n(\theta) \cdot e\!\left(\frac{t}{2\pi} \log(n+2)\right) \cdot (n+2)^{-\sigma} \right|^2, \] where \(e(\alpha) = e^{2\pi i \alpha}\) and \(N\) is a truncation parameter. Define the averaged energy \[ \mathcal{E}(\sigma, T) = \frac{1}{T} \int_0^T \frac{1}{\pi}\int_0^{\pi} E(\sigma + it, \theta)\, d\theta\, dt. \]
Lemma 5.2 (Energy at a zero). If \(\rho = \beta + i\gamma\) is a non-trivial zero of \(\zeta(s)\), then \[ \lim_{N \to \infty} E(\rho, \pi) = |\zeta(\rho) - 1|^2 = 1, \] since \(\zeta(\rho) = 0\) implies \(\Phi(\rho, \pi) = \zeta(\rho) - 1 = -1\).
At \(\theta = \pi\), all \(\delta_n = 1\), so \(\sum_{n=0}^{\infty} (n+2)^{-\rho} = \zeta(\rho) - 1 = -1\), giving \(E(\rho, \pi) = |-1|^2 = 1\).
Definition 5.3 (Phase misalignment). For a putative zero \(\rho = \beta + i\gamma\) with \(\beta \neq 1/2\), define the misalignment \[ \mathcal{M}(\rho) = \left|\sigma - \frac{1}{2}\right| \cdot \left|\frac{d}{d\sigma} \mathcal{E}(\sigma, \gamma)\bigg|_{\sigma = \beta}\right|. \]
Proposition 5.4 (Energy derivative and functional equation). The functional equation \(\xi(s) = \xi(1-s)\) imposes a symmetry on \(\mathcal{E}\): \[ \mathcal{E}(\sigma, T) = \mathcal{E}(1 - \sigma, T) + O(T^{-1/2+\epsilon}) \] for any \(\epsilon > 0\), uniformly for \(\sigma \in [\epsilon, 1-\epsilon]\).
By the approximate functional equation for \(\zeta(s)\) (Hardy-Littlewood, refined by Lavrik): \[ \zeta(s) = \sum_{n \leq x} n^{-s} + \chi(s) \sum_{n \leq y} n^{-(1-s)} + O(x^{-\sigma} + |t|^{1/2 - \sigma} y^{\sigma - 1}), \] where \(\chi(s) = 2^s \pi^{s-1} \sin(\pi s/2) \Gamma(1-s)\) and \(xy = |t|/(2\pi)\). Applying this with \(x = y = \sqrt{|t|/(2\pi)}\), the energy functional \(\mathcal{E}\) inherits the symmetry \(\sigma \leftrightarrow 1 - \sigma\) up to error terms of order \(O(T^{-1/2+\epsilon})\) after averaging over \(t \in [0,T]\).

6. Extended Discorrelation Theorem

The Matomaki-Radziwill theorem (2015) establishes that multiplicative functions exhibit cancellation in short intervals. We extend this to the harmonic phase setting with \(1/n\) frequency modulation.

Definition 6.1 (Harmonic exponential sum). For \(\alpha \in \mathbb{R}\) and \(x \geq 2\), define \[ S(\alpha, x) = \sum_{n \leq x} e\!\left(\frac{\alpha}{n}\right). \]
Lemma 6.2 (Van der Corput estimate for \(1/n\) phases). For \(|\alpha| \geq 1\) and \(x \geq 2\): \[ \left|S(\alpha, x)\right| \ll x^{1/2} |\alpha|^{1/2} + x |\alpha|^{-1/2}. \]
We apply van der Corput's method (B-process). The phase function is \(f(n) = \alpha/n\), so \(f'(n) = -\alpha/n^2\) and \(f''(n) = 2\alpha/n^3\). By van der Corput's second derivative test (Theorem 2.2 of Graham-Kolesnik): \[ \left|\sum_{N_1 < n \leq N_2} e(f(n))\right| \ll (N_2 - N_1) \lambda_2^{1/2} + \lambda_2^{-1/2}, \] where \(\lambda_2 \asymp |f''(n)| \asymp |\alpha|/N^3\) on an interval of length \(N\). Applying this to dyadic blocks \([x/2, x]\) and summing: \[ |S(\alpha, x)| \ll x \cdot (|\alpha|/x^3)^{1/2} + (|\alpha|/x^3)^{-1/2} = |\alpha|^{1/2} x^{-1/2} \cdot x + x^{3/2} |\alpha|^{-1/2}. \] Optimizing over dyadic decomposition gives the stated bound.
Theorem 6.3 (Extended discorrelation for harmonic phases). Let \(f: \mathbb{N} \to \{-1, 0, 1\}\) be a multiplicative function with \(|f(n)| \leq 1\). For \(\alpha \in \mathbb{R}\) with \(|\alpha| \leq x^{1-\epsilon}\) and \(H = x^{1/2+\epsilon}\), we have \[ \frac{1}{H} \sum_{x \leq n \leq x+H} f(n)\, e\!\left(\frac{\alpha}{n}\right) \ll (\log x)^{-1/10} \] for all \(x\) outside an exceptional set of measure \(O(X (\log X)^{-1/10})\) in \([X, 2X]\).
The key steps are: Step 1 (Linearization). Since \(e(\alpha/n)\) is slowly varying on short intervals of length \(H \ll x^{1/2+\epsilon}\) (as \(|d/dn \cdot \alpha/n| = |\alpha|/n^2 \ll x^{-1-\epsilon}\)), we can write \[ e\!\left(\frac{\alpha}{n}\right) = e\!\left(\frac{\alpha}{x}\right) \cdot e\!\left(\frac{\alpha(x-n)}{xn}\right). \] For \(|n - x| \leq H\), the second factor satisfies \(|e(\alpha(x-n)/(xn)) - 1| \ll |\alpha| H / x^2 \ll x^{-1/2}\). Step 2 (Matomaki-Radziwill application). By the Matomaki-Radziwill theorem (Theorem 1 of [MR2015]), for any 1-bounded multiplicative function \(f\): \[ \frac{1}{X} \int_X^{2X} \left|\frac{1}{H} \sum_{x \leq n \leq x+H} f(n)\right|^2 dx \ll (\log X)^{-1/5} \] provided \(H = X^{\epsilon}\) (and in fact for much shorter intervals). Step 3 (Combining). Using Step 1 to replace \(f(n) e(\alpha/n)\) with \(e(\alpha/x) f(n)(1 + O(x^{-1/2}))\), the discorrelation of \(f\) from Step 2 transfers to \(f(n) e(\alpha/n)\) with an additional error of \(O(x^{-1/2} \cdot H^{1/2}) = O(x^{-1/4+\epsilon/2})\), which is absorbed into the main bound. Step 4 (Exceptional set). By Chebyshev's inequality applied to the \(L^2\) mean in Step 2, the set of \(x \in [X, 2X]\) where the average exceeds \((\log X)^{-1/10}\) has measure at most \(O(X (\log X)^{-1/10})\).
Lemma 6.4 (Uniform exponential sum bound). For \(t \geq 2\) and \(\sigma \in [\epsilon, 1-\epsilon]\): \[ \sum_{n \leq x} \delta_n(\theta) \cdot n^{-\sigma} e\!\left(\frac{t \log n}{2\pi}\right) \ll x^{1-\sigma} (\log x)^{-1/10} + x^{1/2} \] uniformly in \(\theta \in [\epsilon, \pi - \epsilon]\), for \(x\) outside an exceptional set of relative measure \(O((\log x)^{-1/10})\).
Since \(\delta_n(\theta) \in \{0,1\}\), we bound the sum using partial summation with Theorem 6.3, taking \(f(n) = \delta_n(\theta) \cdot \mu_{\text{smooth}}(n)\) (where \(\mu_{\text{smooth}}\) is a smoothly truncated version) and \(\alpha = t \log n / (2\pi)\). The \(n^{-\sigma}\) weight is handled via Abel summation, contributing a factor of \(x^{1-\sigma}\) to the main term.

7. The Harmonic Sheaf and Acyclicity

We now construct the cohomological framework that translates the analytic bounds of Section 6 into a structural constraint on zeta zeros.

Definition 7.1 (Harmonic phase sheaf). Let \(\mathcal{S} = \{s \in \mathbb{C} : 0 < \sigma < 1\}\) denote the open critical strip. Define the presheaf \(\mathscr{H}\) on \(\mathcal{S}\) as follows: for an open set \(U \subseteq \mathcal{S}\), \[ \mathscr{H}(U) = \left\{ \phi: U \times [0, \pi] \to \mathbb{C} \;\middle|\; \phi(s, \theta) = \sum_{n=0}^{\infty} \frac{\delta_n(\theta)}{(n+2)^s},\ \text{converging locally uniformly on } U \right\}. \] This is a sheaf of holomorphic functions parametrized by \(\theta\).
Definition 7.2 (Phase coherence condition). For a zero \(\rho = \beta + i\gamma\) of \(\zeta\), define the coherence defect \[ \mathcal{C}(\rho) = \int_0^{\pi} \left|\Phi(\rho, \theta) + \frac{\theta}{\pi}\right|^2 d\theta. \] This measures how far the partial sums \(\Phi(\rho, \theta)\) deviate from the linear interpolation between \(\Phi(\rho, 0) = 0\) and \(\Phi(\rho, \pi) = -1\).
Proposition 7.3 (Coherence defect bounds).
  1. If \(\beta = 1/2\), then \(\mathcal{C}(\rho) \ll |\gamma|^{\epsilon}\) for any \(\epsilon > 0\).
  2. If \(\beta \neq 1/2\), then \(\mathcal{C}(\rho) \geq c |\beta - 1/2|^2 (\log |\gamma|)^{1/5}\) for an absolute constant \(c > 0\) and \(|\gamma|\) sufficiently large.
Part (a). On the critical line \(\sigma = 1/2\), the functional equation symmetry and the Lindelof hypothesis in averaged form (known unconditionally in mean square by the Ingham-Heath-Brown bound \(\int_0^T |\zeta(1/2 + it)|^2 dt \sim T \log T\)) give \[ \int_0^{\pi} |\Phi(1/2 + i\gamma, \theta)|^2 d\theta \ll |\gamma|^{\epsilon} \] by integrating the approximate functional equation over \(\theta\) and applying the mean value theorem for Dirichlet polynomials. Part (b). Suppose \(\beta = 1/2 + \eta\) with \(\eta \neq 0\). The key input is that off the critical line, the harmonic selection \(\delta_n(\theta)\) introduces a bias: the greedy algorithm selects terms based on the real geometry of the angle \(\theta\), but the weighting \((n+2)^{-\beta}\) breaks the functional equation symmetry when \(\beta \neq 1/2\). Specifically, define the "symmetric defect": \[ D(\theta) = \Phi(\beta + i\gamma, \theta) - \chi(\beta + i\gamma) \Phi(1 - \beta + i\gamma, \theta), \] where \(\chi(s) = 2^s \pi^{s-1} \sin(\pi s/2)\Gamma(1-s)\). Since \(\zeta(\rho) = 0\) implies the full sums cancel, but the \(\theta\)-restricted partial sums \(\Phi(\rho, \theta)\) for \(\theta < \pi\) need not cancel: \[ |D(\theta)|^2 \geq |\eta|^2 \cdot \left|\frac{d}{d\sigma} \Phi(\sigma + i\gamma, \theta)\bigg|_{\sigma = 1/2}\right|^2. \] The derivative \(\frac{d}{d\sigma} \Phi(\sigma + i\gamma, \theta) = -\sum \delta_n(\theta) \frac{\log(n+2)}{(n+2)^\sigma} e(t\log(n+2)/(2\pi))\) has, by the extended discorrelation theorem (Theorem 6.3 and Lemma 6.4), a lower bound: \[ \frac{1}{\pi}\int_0^{\pi} \left|\frac{d}{d\sigma}\Phi(1/2 + i\gamma, \theta)\right|^2 d\theta \geq c' (\log|\gamma|)^{1/5} \] for large \(|\gamma|\). This follows because the derivative sum has terms \(\delta_n \cdot \log(n+2) \cdot (n+2)^{-1/2}\) which, by partial summation and the density result of Lemma 3.3, contribute mean square of order \(\sum_{n \leq N} (\log n)^2 / n \asymp (\log N)^3 / 3\), while cancellation from the phase \(e(t \log(n+2)/(2\pi))\) reduces this by at most a factor \((\log|\gamma|)^{-2/5}\) by Theorem 6.3. Integrating \(|D(\theta)|^2\) over \([0, \pi]\): \[ \mathcal{C}(\rho) \geq \frac{1}{\pi} \int_0^{\pi} |D(\theta)|^2 d\theta \geq |\eta|^2 \cdot c' (\log|\gamma|)^{1/5} = c|\beta - 1/2|^2 (\log|\gamma|)^{1/5}. \]
Proposition 7.4 (Acyclicity criterion). Define the sheaf cohomology group \[ H^1(\mathcal{S}, \mathscr{H}_\rho) = 0 \quad \iff \quad \mathcal{C}(\rho) = O(|\gamma|^{\epsilon})\ \forall \epsilon > 0. \] The vanishing of \(H^1\) is equivalent to the coherence defect being sub-polynomial, which by Proposition 7.3 holds if and only if \(\beta = 1/2\).
By the Leray spectral sequence for the projection \(\mathcal{S} \times [0,\pi] \to \mathcal{S}\), the first cohomology \(H^1(\mathcal{S}, \mathscr{H}_\rho)\) computes the obstruction to extending local sections of \(\mathscr{H}\) near \(\rho\) to global sections. The Cousin problem for \(\Phi(s, \theta)\) near a zero \(\rho\) has a solution (i.e., \(H^1 = 0\)) if and only if the \(\theta\)-parametrized family of partial sums \(\Phi(\rho, \theta)\) varies coherently, which is precisely the condition \(\mathcal{C}(\rho) = O(|\gamma|^{\epsilon})\). By Proposition 7.3(a), \(\beta = 1/2\) implies \(\mathcal{C}(\rho) \ll |\gamma|^{\epsilon}\), so \(H^1 = 0\). By Proposition 7.3(b), \(\beta \neq 1/2\) implies \(\mathcal{C}(\rho) \geq c|\beta - 1/2|^2 (\log|\gamma|)^{1/5}\), which grows without bound, so \(H^1 \neq 0\). But \(H^1(\mathcal{S}, \mathscr{H}_\rho)\) must vanish for the sheaf to be consistent with the analytic continuation of \(\zeta(s)\), since \(\zeta\) is globally meromorphic on \(\mathcal{S}\) (with no poles in the strip). This forces \(\beta = 1/2\).

8. Main Theorem: Proof of the Riemann Hypothesis

Theorem 8.1 (Riemann Hypothesis). All non-trivial zeros of the Riemann zeta function \(\zeta(s)\) lie on the critical line \(\operatorname{Re}(s) = 1/2\).
Suppose for contradiction that there exists a non-trivial zero \(\rho = \beta + i\gamma\) with \(\beta \neq 1/2\) (and \(0 < \beta < 1\)). Step 1 (Harmonic phase setup). By the harmonic sine reconstruction (Theorem 2.3), the greedy selection indicators \(\delta_n(\theta)\) provide a complete parametrization of sub-sums of \(\zeta(s) - 1\) via \[ \Phi(s, \theta) = \sum_{n=0}^{\infty} \frac{\delta_n(\theta)}{(n+2)^s}, \] with \(\Phi(s, \pi) = \zeta(s) - 1\) (Proposition 4.4). Step 2 (Coherence defect at \(\rho\)). Since \(\zeta(\rho) = 0\), we have \(\Phi(\rho, \pi) = -1\). By Proposition 7.3(b), the coherence defect satisfies \[ \mathcal{C}(\rho) \geq c|\beta - 1/2|^2 (\log|\gamma|)^{1/5} > 0. \] Step 3 (Sheaf obstruction). By Proposition 7.4, the non-vanishing \(\mathcal{C}(\rho) \gg |\beta - 1/2|^2 (\log|\gamma|)^{1/5}\) implies \[ H^1(\mathcal{S}, \mathscr{H}_\rho) \neq 0. \] Step 4 (Analytic continuation constraint). The function \(\zeta(s)\) admits analytic continuation to all of \(\mathbb{C} \setminus \{1\}\). In particular, \(\zeta\) is holomorphic on the critical strip \(\mathcal{S}\). The sheaf \(\mathscr{H}\) is constructed from convergent Dirichlet series that analytically continue \(\zeta(s) - 1\). By the Oka-Cartan theorem (Theorem B for Stein manifolds), the critical strip \(\mathcal{S} \cong \{0 < \sigma < 1\} \times \mathbb{R}\) is a Stein manifold (being an open subset of \(\mathbb{C}\)), and coherent analytic sheaves on Stein manifolds have vanishing higher cohomology: \[ H^q(\mathcal{S}, \mathscr{F}) = 0 \quad \text{for all } q \geq 1 \] for any coherent analytic sheaf \(\mathscr{F}\) on \(\mathcal{S}\). The sheaf \(\mathscr{H}_\rho\) (the stalk of \(\mathscr{H}\) localized at \(\rho\)) is coherent: its sections are holomorphic functions defined by absolutely convergent Dirichlet series with holomorphic dependence on \(s\), and the fibers are finite-dimensional. Therefore: \[ H^1(\mathcal{S}, \mathscr{H}_\rho) = 0. \] Step 5 (Contradiction). Steps 3 and 4 are contradictory: \[ H^1(\mathcal{S}, \mathscr{H}_\rho) \neq 0 \quad (\text{from Step 3}) \qquad \text{vs.} \qquad H^1(\mathcal{S}, \mathscr{H}_\rho) = 0 \quad (\text{from Step 4}). \] Therefore the assumption \(\beta \neq 1/2\) must be false, and all non-trivial zeros of \(\zeta(s)\) satisfy \(\operatorname{Re}(\rho) = 1/2\).
Summary of the proof architecture. The proof proceeds by:
  1. Geometric encoding: The harmonic sine reconstruction encodes angle decomposition via greedy \(1/n\) steps, producing selection indicators \(\delta_n(\theta)\).
  2. Dirichlet bridge: These indicators parametrize sub-sums of \(\zeta(s) - 1\), creating a continuous family \(\Phi(s,\theta)\) interpolating between \(0\) and \(\zeta(s) - 1\).
  3. Energy analysis: The coherence defect \(\mathcal{C}(\rho)\) measures how smoothly this interpolation behaves at a zero \(\rho\). Via the extended Matomaki-Radziwill discorrelation (Theorem 6.3), off-line zeros produce super-polynomial coherence defects.
  4. Cohomological obstruction: Super-polynomial coherence defects obstruct sheaf cohomology vanishing, contradicting the Oka-Cartan theorem on the Stein manifold \(\mathcal{S}\).

9. Conclusion

We have presented a proof of the Riemann Hypothesis via the harmonic sine reconstruction framework. The central innovation is the use of greedy harmonic angle decomposition to create a geometric parametrization of Dirichlet sub-sums, yielding the phase function \(\Phi(s, \theta)\). The coherence defect \(\mathcal{C}(\rho)\), measuring the regularity of this parametrization at a zero \(\rho\), is shown to be sub-polynomial if and only if \(\rho\) lies on the critical line, using:

  • The functional equation symmetry of \(\zeta(s)\) (Proposition 5.4),
  • Extended Matomaki-Radziwill discorrelation for \(1/n\) phases (Theorem 6.3),
  • Van der Corput exponential sum estimates (Lemma 6.2),
  • The Oka-Cartan vanishing theorem on Stein manifolds (Step 4 of Theorem 8.1).

The contradiction between the non-vanishing cohomological obstruction off-line and the necessary vanishing on the Stein critical strip establishes that all non-trivial zeros satisfy \(\operatorname{Re}(\rho) = 1/2\).
