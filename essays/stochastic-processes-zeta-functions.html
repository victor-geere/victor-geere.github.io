Introduction

Stochastic processes, which model systems with random changes, can be linked to zeta functions—mathematical functions central to number theory, like the Riemann zeta function. This connection is especially evident in p-adic analysis, a field using a different number system, and through advanced models involving supersymmetry. This exploration delves into how these concepts intertwine, hypothesizing new properties for further analysis.

While statistics gives us a means to approximate the behaviour of stochastic processes it does not give as an algebraic handle on the individual data points or make stochastic processes invertable to their generating functions. Expressing a stochastic process in terms of a Zeta function might make that possible as we can deterministically calculate each summand of a Zeta function if the input is known. In this document we explore the connection between stochastic processes such as the Ornstein-Uhlenbeck process and Zeta functions.

For example, the Riemann zeta function’s relation, \(\int_0^\infty \frac{x^{s-1}}{e^x - 1} dx = \Gamma(s) \zeta(s)\), suggests that process densities’ transforms could mirror zeta function properties.
