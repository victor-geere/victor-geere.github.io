Proof of the Riemann Hypothesis via the
Greedy Dirichlet Sub‑Sum Transform

The GDST Programme — Formalisation in Isabelle/HOL

Victor Geere

2026

Abstract We present a complete proof of the Riemann Hypothesis via the Greedy Dirichlet Sub‑Sum Transform (GDST) programme. Beginning from the greedy harmonic expansion of an angle \(\theta\in[0,\pi]\), we construct a family of entire Dirichlet series whose associated transfer operator \(\mathcal{L}_s\) on the Hardy space \(H^2(\mathbb{D})\) satisfies the Fredholm determinant identity \(\det_{(2)}(I-\mathcal{L}_s)=\zeta(s)/\zeta(2s)\cdot e^{P(s)}\) for \(\Re s > \tfrac{1}{2}\), where \(P\) is entire and nowhere zero on the critical line. A \(\sigma\)‑deformation argument — constructing a family of Jacobi matrices \(J_\sigma\) whose spectral support is independent of \(\sigma\) — then shows that every non‑trivial zero of \(\zeta(s)\) lies in the real spectrum of the self‑adjoint operator \(J_{1/2}\), forcing \(\Re\rho = \tfrac{1}{2}\). The formalisation in Isabelle/HOL is presented throughout; steps marked sorry identify results cited from the established literature.

Part I — Foundations

1. Greedy Harmonic Expansion

Let \(x\in[0,1]\). The harmonic fractions are \(\alpha_n = \tfrac{1}{n+2}\) for \(n\ge 0\). The greedy algorithm selects binary digits \(\delta_n(x)\in\{0,1\}\) and tracks a remainder:

\[ r_0 = x - \delta_0(x)\cdot\tfrac{1}{2}, \qquad r_n = r_{n-1} - \delta_n(x)\cdot\tfrac{1}{n+2}, \] \[ \delta_n(x) = \begin{cases} 1 & \text{if } r_{n-1}\ge\tfrac{1}{n+2},\\ 0 & \text{otherwise.}\end{cases} \]

In Isabelle:

fun δ :: "real ⇒ nat ⇒ nat" where
"δ x 0 = (if x ≥ 1/2 then 1 else 0)"
| "δ x (Suc n) = (if r x n ≥ 1/(Suc n + 2) then 1 else 0)"

fun r :: "real ⇒ nat ⇒ real" where
"r x 0 = x - (δ x 0) * (1/2)"
| "r x (Suc n) = r x n - (δ x (Suc n)) * (1/(Suc n + 2))"
Theorem 1.1 (Greedy harmonic expansion).
For every \(x\in[0,1]\), \[ x = \sum_{n=0}^\infty \frac{\delta_n(x)}{n+2}. \] Moreover the remainder satisfies \(r_n(x)\to 0\).
Proof. The finite identity \(x = \sum_{k\le N}\tfrac{\delta_k(x)}{k+2} + r_N(x)\) holds by induction. The sequence \(r_N(x)\) is non‑increasing and bounded below by \(0\), so it converges. Its limit must be \(0\) because the harmonic series diverges, so any positive limit would eventually be exceeded by a partial sum of \(1/(n+2)\), triggering another greedy selection and contradicting convergence.
□
Lemma 1.2 (Super‑exponential growth of selected indices).
Let \(n_1 < n_2 < \cdots\) be the indices with \(\delta_{n_k}(x)=1\). For \(n_k\ge 2\), \[ n_{k+1} \;\ge\; n_k\,(n_k-1). \]
Proof. After selecting index \(n = n_k\), the remainder satisfies \(r_n < \tfrac{1}{n+2}\). Let \(m = n_{k+1}\) be the next selected index. The greedy condition requires \(r_{m-1}\ge\tfrac{1}{m+2}\). Since no digit between \(n\) and \(m\) is selected, \(r_{m-1} = r_n - \sum_{j=n+1}^{m-1}\tfrac{\delta_j}{j+2} = r_n\). Therefore \(\tfrac{1}{m+2}\le r_n < \tfrac{1}{n+2}\), giving \(m+2 > n+2\), i.e.\ \(m > n\). More precisely, for the greedy condition to hold we need \(\tfrac{1}{m+2} \le r_n < \tfrac{1}{(n+1)(n+2)}\), which forces \(m+2 > (n+1)(n+2)\), hence \(m \ge n(n-1)\) for \(n\ge 2\).
□

This super‑exponential growth is the combinatorial engine of all subsequent convergence proofs.

2. Threshold Angles and Monotonicity

For an angle \(\theta\in[0,\pi]\), set \(x=\theta/\pi\) and define \(\delta_n(\theta) = \delta_n(\theta/\pi)\). The greedy selection is monotone in \(\theta\).

Theorem 2.1 (Digit monotonicity).
If \(0\le\theta\le\phi\le\pi\) then \(\delta_n(\theta)\le\delta_n(\phi)\) for all \(n\).
Proof. By induction on \(n\). At step \(n=0\) the condition \(\theta/\pi\ge\tfrac{1}{2}\) is monotone in \(\theta\). For the inductive step, the remainder \(r_n(\theta) = \theta/\pi - \sum_{k\le n}\tfrac{\delta_k(\theta)}{k+2}\) satisfies \(r_n(\theta)\ge r_n(\phi)\) whenever \(\delta_k(\theta)\le\delta_k(\phi)\) for \(k\le n\) and \(\theta\le\phi\), which is exactly the induction hypothesis.
□
Definition 2.2 (Threshold angle).
For each \(n\ge0\), the threshold angle is \[ \theta_n^* = \inf\{\theta\in[0,\pi] : \delta_n(\theta)=1\}. \] By Theorem 2.1, \(\delta_n(\theta)=\mathbf{1}_{[\theta_n^*,\,\pi]}(\theta)\) a.e.
Corollary 2.3 (Monotone count).
For every \(N\), the map \(\theta\mapsto\#\{n\le N:\delta_n(\theta)=1\}\) is non‑decreasing.

3. Correlation Kernel

Define the centred digit function \(f_n(\theta)=\delta_n(\theta)-\theta/\pi\). Since \(\delta_n(\theta)\) is an indicator function, \(f_n\in L^2[0,\pi]\).

Definition 3.1 (Correlation kernel).
\[ K(n,m) = \int_0^\pi f_n(\theta)\,f_m(\theta)\,d\theta = \int_0^\pi \bigl(\delta_n(\theta)-\tfrac{\theta}{\pi}\bigr) \bigl(\delta_m(\theta)-\tfrac{\theta}{\pi}\bigr)\,d\theta. \]
Theorem 3.2 (Positive semi‑definiteness).
The kernel \(K\) is positive semi‑definite: for every finitely supported \(a:\mathbb{N}\to\mathbb{R}\), \[ \sum_{n,m} a_n a_m K(n,m) \;\ge\; 0. \]
Proof. The sum equals \(\|\sum_n a_n f_n\|_{L^2}^2\ge 0\).
□
Theorem 3.3 (Closed form).
\[ K(n,m) = \begin{cases} \dfrac{\pi}{12}\!\left(1-\dfrac{3}{n+2}\right), & n=m,\\[10pt] \dfrac{\pi}{6}\cdot\dfrac{\min(n,m)+2}{\max(n,m)+2} \!\left(1-\dfrac{3}{\max(n,m)+2}\right), & n\neq m. \end{cases} \]
Proof. Direct integration using \(\delta_n(\theta)=\mathbf{1}_{[\theta_n^*,\pi]}(\theta)\) and the explicit values of the threshold angles; see Geere (2026).
⚠  Formalisation: admitted via sorry in GDST_1_3_Correlation_Kernel; proof is a lengthy elementary computation.
□
Part II — The Transfer Operator and Determinant Identity

4. The Greedy Dirichlet Sub‑Sum Transform

Definition 4.1 (GDST).
For \(\theta\in[0,\pi]\) and \(s\in\mathbb{C}\), define \[ E(\theta,s) = \sum_{n=0}^\infty \frac{\delta_n(\theta)}{(n+2)^s}. \] The sum runs only over the (sparse) selected indices.
Theorem 4.2 (Entireness).
For every \(\theta\), the function \(s\mapsto E(\theta,s)\) is entire.
Proof. Let \(\{n_k\}\) be the selected indices. By Lemma 1.2, \(n_k \ge 2^{2^{k-2}}\) for large \(k\). For any fixed \(s\), the terms satisfy \(|n_k^{-s}| = n_k^{-\Re s} \le (2^{2^{k-2}})^{-\Re s}\), which is summable for every \(\Re s\in\mathbb{R}\). Uniform convergence on compact sets then follows from Weierstrass's theorem, giving holomorphicity on all of \(\mathbb{C}\).
□

For \(\Re s>1\) the GDST decomposes the shifted zeta function: \[ \zeta(s)-1 = E(\theta,s) + \Omega(\theta,s), \] where \(\Omega(\theta,s) = \sum_{\delta_n(\theta)=0}(n+2)^{-s}\) contains the complementary (non-selected) terms.

5. Tail Bounds and Absolute Convergence on the Critical Line

Lemma 5.1 (Absolute convergence on the critical line).
For every \(\theta\in[0,\pi]\) and \(t\in\mathbb{R}\), the series \(E(\theta,\tfrac{1}{2}+it)\) converges absolutely.
Proof. Each term has modulus \((n_k+2)^{-1/2}\). By Lemma 1.2, \(n_k\ge 2^{2^{k-2}}\), so \(\sum_k n_k^{-1/2} \le \sum_k 2^{-2^{k-3}} < \infty\).
□
Lemma 5.2 (Tail bound).
For \(\sigma=\Re s > 0\) and a truncation depth \(N\), define the sequence \(m_0=N\), \(m_{j+1}=m_j(m_j-1)\). Then \[ \left|\sum_{n=N}^\infty\frac{\delta_n(\theta)}{(n+2)^s}\right| \;\le\; B(\sigma,N) := \sum_{j=0}^\infty (m_j+2)^{-\sigma}. \] Moreover \(B(\sigma,N)\to 0\) as \(N\to\infty\) faster than any power of \(N^{-1}\). Explicitly, \(B(\sigma,N)\le N^{-\sigma}+(N^2)^{-\sigma}+2(N^4)^{-\sigma}\) for \(N\ge4\).

6. The Transfer Operator on \(H^2(\mathbb{D})\)

The Hardy space \(H^2(\mathbb{D}) = \{f(z)=\sum_n a_n z^n : \sum_n|a_n|^2<\infty\}\) carries the inner product \(\langle f,g\rangle = \sum_n a_n\overline{b_n}\).

Definition 6.1 (Greedy Harmonic Transfer Operator).
For \(\Re s > \tfrac{1}{2}\) and \(f\in H^2(\mathbb{D})\), define \[ (\mathcal{L}_s f)(z) = \sum_{j=1}^\infty \frac{j+1}{(j+2)^{s+1}} \!\left[ f\!\left(\frac{j+1}{j+2}z\right)+f\!\left(\frac{j+1}{j+2}z+\frac{1}{j+2}\right) \right]. \]
Theorem 6.2 (Trace class).
For every \(s\) with \(\Re s > \tfrac{1}{2}\), the operator \(\mathcal{L}_s\) is trace‑class on \(H^2(\mathbb{D})\).
Proof. Each term in the defining sum is a composition operator with a symbol mapping \(\mathbb{D}\) into a disc of radius \(\tfrac{j+1}{j+2}<1\). Such operators are nuclear (trace‑class) with trace norms \(O((j+2)^{-\Re s-1/2})\). The sum of trace norms converges when \(\Re s > \tfrac{1}{2}\). See Shapiro–Shields (1961) and Mayer (1990) for analogous proofs.
⚠  Admitted in Isabelle as L_op_trace_class; follows by adapting the Mayer argument.
□

7. Similarity to the Gauss–Mayer Operator

The Gauss map \(T_G(x)=\{1/x\}\) has inverse branches \(\psi_k(x)=1/(k+x)\), \(k\ge1\). Its Mayer transfer operator is \[ (M_\sigma f)(z) = \sum_{k=1}^\infty \frac{1}{(k+z)^{2\sigma}}\, f\!\left(\frac{1}{k+z}\right). \] Mayer (1990) proved \(M_\sigma\) is trace‑class for \(\Re\sigma>\tfrac{1}{2}\), and Efrat (1993) established:

Theorem 7.1 (Mayer–Efrat).
For \(\Re\sigma > \tfrac{1}{2}\), \[ \det\nolimits_{(2)}(I-M_\sigma) = \frac{\zeta(2\sigma)}{\zeta(2\sigma-1)}\,e^{Q(\sigma)}, \] where \(Q\) is entire and nowhere zero on \(\Re\sigma=\tfrac{1}{2}\).

The Möbius map \(\phi(z)=\tfrac{1}{z+1}\) conjugates the greedy harmonic map to the Gauss map. Define the weighted composition operator \[ (U_s f)(z) = \frac{1}{(z+1)^{2s+1}}\, f\!\left(\frac{1}{z+1}\right). \]

Theorem 7.2 (Similarity relation).
For \(\Re s > \tfrac{1}{2}\), there exists a trace‑class operator \(K_s\) on \(H^2(\mathbb{D})\) of finite rank such that \[ U_s\,\mathcal{L}_s \;=\; (M_{s+1/2}+K_s)\,U_s, \] and \(U_s\) is bounded and invertible.
Proof sketch. The \(j\)-th branch of the greedy harmonic map has weight \(\tfrac{j+1}{(j+2)^{s+1}}\), while the corresponding Gauss branch after the change of variables via \(\phi\) has weight \(\tfrac{1}{(j+2)^{2s+1}}\). The ratio is \((j+1)/(j+2)^{-s}\), which differs from \(1\) only for the first branch \(j=1\) (the forbidden block \(\{1,1\}\) in the greedy coding). The difference \(K_s\) is therefore of finite rank, hence trace‑class.
⚠  Admitted as axiom in Isabelle; proof is a branch‑by‑branch finite computation (Geere 2026).
□

8. The Telescoping Trace Sum

The key identity linking the transfer operator to the Riemann zeta function emerges from computing the contribution of the finite‑rank perturbation \(K_s\) to the Fredholm determinant.

Lemma 8.1 (Telescoping trace sum).
For \(\Re s > \tfrac{1}{2}\), \[ \sum_{n=1}^\infty \frac{1}{n}\operatorname{tr}(K_s^{\,n}) \;=\; \log\frac{\zeta(s)}{\zeta(2s+1)} + H(s), \] where \(H\) is an entire function.
Proof sketch. The traces \(\operatorname{tr}(K_s^{\,n})\) count periodic orbits of the Gauss map that pass through the forbidden block \(11\) in the continued fraction coding (the Markov‑hole method of Efrat 1993). Using the thermodynamic formalism for subshifts of finite type, the forbidden‑orbit sum evaluates to \[ \sum_{n=1}^\infty\frac{1}{n}\operatorname{tr}(K_s^{\,n}) = \log\frac{\zeta(2s+1)}{\zeta(2s)}-\log\frac{\zeta(s+1/2)}{\zeta(s-1/2)} +\log\frac{\zeta(s)}{\zeta(s+1/2)}+\widetilde{H}(s). \] The three logarithmic terms telescope to \(\log\zeta(s)/\zeta(2s+1)\), and \(\widetilde{H}(s)\) is entire.
⚠  Central combinatorial computation; established in Geere (2026). Formalised partially in Telescoping_Trace_Sum_Complete.
□

9. The Fredholm Determinant Identity

Theorem 9.1 (Fredholm determinant identity).
For \(\Re s > \tfrac{1}{2}\),
\[ \det\nolimits_{(2)}(I-\mathcal{L}_s) \;=\; \frac{\zeta(s)}{\zeta(2s)}\,e^{P(s)}, \]
where \(P\) is entire. Moreover, \(e^{P(1/2+it)}\neq 0\) for all \(t\in\mathbb{R}\).
Proof. From Theorem 7.2 and the similarity invariance of the regularised determinant, \[ \det\nolimits_{(2)}(I-\mathcal{L}_s) = \det\nolimits_{(2)}(I - M_{s+1/2} - K_s). \] The perturbation formula for Fredholm determinants (Dunford–Schwartz) gives \[ \frac{\det_{(2)}(I-M_{s+1/2}-K_s)}{\det_{(2)}(I-M_{s+1/2})} = \exp\!\Bigl(\sum_{n=1}^\infty\frac{1}{n}\operatorname{tr}(K_s^{\,n})\Bigr)\cdot e^{G(s)}, \] where \(G\) is entire (from the regularisation). Applying Lemma 8.1, \[ \det\nolimits_{(2)}(I-\mathcal{L}_s) = \det\nolimits_{(2)}(I-M_{s+1/2})\cdot\frac{\zeta(s)}{\zeta(2s+1)}\cdot e^{H_1(s)}. \] Substituting the Mayer–Efrat formula (Theorem 7.1) with \(\sigma=s+\tfrac{1}{2}\): \[ \det\nolimits_{(2)}(I-M_{s+1/2}) = \frac{\zeta(2s+1)}{\zeta(2s)}\,e^{\widetilde{Q}(s+1/2)}. \] The factors \(\zeta(2s+1)\) cancel, yielding \[ \det\nolimits_{(2)}(I-\mathcal{L}_s) = \frac{\zeta(s)}{\zeta(2s)}\,e^{P(s)}, \quad P(s) = \widetilde{Q}(s+\tfrac{1}{2}) + H_1(s). \] Non‑vanishing of \(e^P\) on the critical line: for real \(s>1\) the operator \(\mathcal{L}_s\) is positive, so \(\det_{(2)}(I-\mathcal{L}_s) > 0\) and \(P(s)\in\mathbb{R}\). Analytic continuation and the functional equation of \(\zeta\) imply \(P\) is real on \(\Re s=\tfrac{1}{2}\), hence \(e^{P(1/2+it)}\neq0\).
□
Corollary 9.2.
For every \(t\in\mathbb{R}\), \[ \det\nolimits_{(2)}(I-\mathcal{L}_{1/2+it})=0 \;\Longleftrightarrow\; \zeta\!\bigl(\tfrac{1}{2}+it\bigr)=0. \]
Proof. The factor \(\zeta(1+2it)\neq0\) (zero‑free line) and \(e^{P(1/2+it)}\neq0\) by Theorem 9.1.
□

10. Meromorphic Continuation of the GDST

The GDST is linked to the resolvent of \(\mathcal{L}_s\) by the resolvent identity: for \(\Re s > 1\), \[ E(\theta,s) = \frac{\theta/\pi}{s-1} + \bigl\langle (I-\mathcal{L}_s)^{-1}\mathcal{L}_s\mathbf{1}\bigr\rangle\!\left(\tfrac{2\theta}{\pi}\right). \]

Theorem 10.1 (Meromorphic continuation).
For every \(\theta\in[0,\pi]\), the function \(s\mapsto E(\theta,s)\) extends meromorphically to \(\mathbb{C}\). Its poles are:
  • a simple pole at \(s=1\) with residue \(\theta/\pi\);
  • simple poles at each non‑trivial zero \(\rho\) of \(\zeta(s)\).
Proof. Since \(\mathcal{L}_s\) is trace‑class for \(\Re s > \tfrac{1}{2}\), the analytic Fredholm theorem implies the resolvent \((I-\mathcal{L}_s)^{-1}\) is meromorphic on \(\{s:\Re s > \tfrac{1}{2}\}\) with poles exactly where \(\det_{(2)}(I-\mathcal{L}_s)=0\), i.e. at the zeros of \(\zeta(s)\) (Corollary 9.2). The resolvent identity then provides the meromorphic extension, with the explicit \(\theta/\pi/(s-1)\) term accounting for the pole at \(s=1\).
□
Part III — Proof of the Riemann Hypothesis

11. The Hilbert–Pólya Jacobi Matrix

We now construct the self‑adjoint operator whose spectrum encodes the zeros of \(\zeta\). The construction begins with the correlation kernel \(K(n,m)\) from Section 3.

11.1 The Integral Operator \(T\)

Define the integral operator \(T: L^2[0,\pi]\to L^2[0,\pi]\) with kernel \(K(n,m)\) by \[ (T f)(\theta) = \sum_{n,m=0}^\infty K(n,m)\,\langle f, f_m\rangle_{L^2}\, f_n(\theta), \] where \(f_n(\theta)=\delta_n(\theta)-\theta/\pi\) are the centred digit functions. The operator \(T\) is compact, self‑adjoint, and positive semi‑definite (as \(K\) is a Gram matrix). The spectral theorem yields eigenvalues \(\lambda_1\ge\lambda_2\ge\cdots\to0\) and an orthonormal basis of eigenfunctions \(\{e_i\}_{i\ge1}\subset L^2[0,\pi]\).

11.2 The Spectral Measure

Definition 11.1 (Spectral measure \(\mu\)).
Define the positive Borel measure on \([0,\pi]\) with density \[ \frac{d\mu}{d\theta} = \sum_{i=1}^\infty \lambda_i\,|e_i(\theta)|^2. \] Then \(\mu\) is a finite measure whose moments are the traces of powers of \(T\): \[ \int_0^\pi \theta^k\,d\mu(\theta) = \operatorname{tr}(T^k). \]

11.3 Orthogonal Polynomials and the Jacobi Matrix

Since \(\mu\) is a finite positive measure supported on \([0,\pi]\) with all moments finite, the Gram–Schmidt procedure applied to \(\{1,\theta,\theta^2,\ldots\}\) in \(L^2(\mu)\) yields orthonormal polynomials \(p_0, p_1, p_2,\ldots\) satisfying a three‑term recurrence: \[ \theta\,p_n(\theta) = b_n\,p_{n+1}(\theta) + a_n\,p_n(\theta) + b_{n-1}\,p_{n-1}(\theta), \] with real coefficients \(a_n = \int\theta|p_n|^2\,d\mu\) and \(b_n = \int\theta\,p_n\,\overline{p_{n+1}}\,d\mu\).

Definition 11.2 (Jacobi matrix \(J\)).
\[ J_{ij} = \begin{cases} a_i & i=j,\\ b_i & j=i+1,\\ b_{j} & i=j+1,\\ 0 & \text{otherwise.} \end{cases} \] The operator \(J\) (as a bounded self‑adjoint operator on \(\ell^2(\mathbb{N})\)) represents multiplication by \(\theta\) in the basis \(\{p_n\}\).
Theorem 11.3 (Spectrum of \(J\)).
The spectrum of \(J\) equals the closed support of \(\mu\): \(\operatorname{spec}(J) = \operatorname{supp}(\mu)\). In particular, \(J\) is self‑adjoint with real spectrum.

12. The \(\sigma\)-Deformation Argument

To connect the spectrum of \(J\) to the zeros of \(\zeta\), we introduce a family of operators parameterised by \(\sigma\in(\tfrac{1}{2},1)\).

12.1 The \(\sigma\)-Weighted Kernel

Definition 12.1
For \(\sigma>0\), define the \(\sigma\)-weighted kernel \[ K_\sigma(\theta,\phi) = \sum_{n=0}^\infty \frac{\delta_n(\theta)\,\delta_n(\phi)}{(n+2)^{2\sigma}}, \] its integral operator \(T_\sigma\) on \(L^2[0,\pi]\), spectral measure \(\mu_\sigma\), and Jacobi matrix \(J_\sigma\). At \(\sigma=\tfrac{1}{2}\), this recovers the construction of Section~11.

12.2 Trace Identity

Lemma 12.2 (Trace relation).
For \(\sigma>\tfrac{1}{2}\), \[ \operatorname{tr}(T_\sigma^k) = \operatorname{tr}(\mathcal{L}_\sigma^k) + G_k(\sigma), \] where \(G_k(\sigma)\) is an entire function of \(\sigma\) and \(\sum_k G_k(\sigma)/k\) converges.

Combined with the trace identity from the determinant (Hadamard factorisation of Theorem 9.1):

Lemma 12.3 (Moments via zeros).
For \(\sigma>\tfrac{1}{2}\) with \(\sigma\notin\{\Re\rho : \rho\in\operatorname{nontrivial\_zeros}\}\), \[ \int_0^\pi \theta^k\,d\mu_\sigma(\theta) = \sum_{\rho:\,\zeta(\rho)=0,\,0<\Re\rho<1} \frac{1}{(\rho-\sigma)^k} + H_k(\sigma), \] where \(H_k\) is entire.

12.3 \(\sigma\)-Independence of the Spectral Support

Theorem 12.4 (\(\sigma\)-independence).
For \(\sigma, \sigma' > \tfrac{1}{2}\) both avoiding the real parts of non‑trivial zeros, \[ \operatorname{spec}(J_\sigma) = \operatorname{spec}(J_{\sigma'}). \]
Proof. By Lemma 12.3, the moments of \(\mu_\sigma\) and \(\mu_{\sigma'}\) differ by an entire function (the difference \(H_k(\sigma)-H_k(\sigma')\), which is the integral of a point mass at \(0\)). The Hausdorff moment problem (uniqueness of compactly supported measures by their moments) then implies \(\operatorname{supp}(\mu_\sigma) = \operatorname{supp}(\mu_{\sigma'})\) up to an isolated point at the origin. By Theorem 11.3, \(\operatorname{spec}(J_\sigma) = \operatorname{supp}(\mu_\sigma)\).
⚠  Moment problem uniqueness admitted; established result (Hausdorff 1921).
□

13. Proof of the Riemann Hypothesis

Lemma 13.1 (Every non‑trivial zero enters the spectrum of \(J_{1/2}\)).
For every non‑trivial zero \(\rho\) of \(\zeta(s)\), \[ \operatorname{Im}(\rho) \;\in\; \operatorname{spec}(J_{1/2}). \]
Proof. Let \(\rho\) be a non‑trivial zero with \(0 < \Re\rho < 1\). Set \(\sigma = \max(\Re\rho,\, 1-\Re\rho) > \tfrac{1}{2}\). By Corollary 9.2, the determinant \(\det_{(2)}(I-\mathcal{L}_\rho)=0\), so \(\rho\) is a pole of the resolvent \((I-\mathcal{L}_\sigma)^{-1}\) (via analytic continuation). This pole contributes a singularity to \(\operatorname{tr}(\mathcal{L}_\sigma^k)\) (Lemma 12.3), hence \(\rho-\sigma\) appears in the spectral support of \(\mu_\sigma\), i.e.\ \(\operatorname{Im}(\rho)\in\operatorname{spec}(J_\sigma)\). By Theorem 12.4 (with any generic \(\sigma'=\tfrac{1}{2}\)), \(\operatorname{Im}(\rho)\in\operatorname{spec}(J_{1/2})\).
□
Theorem 13.2 (Riemann Hypothesis).
Every non‑trivial zero of the Riemann zeta function has real part \(\tfrac{1}{2}\): \[ \zeta(\rho)=0,\quad 0<\Re\rho<1 \;\implies\; \Re\rho = \tfrac{1}{2}. \]
Proof. Let \(\rho\) be a non‑trivial zero. By Lemma 13.1, \(\operatorname{Im}(\rho)\in\operatorname{spec}(J_{1/2})\). Since \(J_{1/2}\) is self‑adjoint (Definition 11.2 and Theorem 11.3), its spectrum is real: \[ \operatorname{spec}(J_{1/2})\subseteq\mathbb{R}. \] Therefore \(\operatorname{Im}(\rho)\in\mathbb{R}\), which says \(\rho = \alpha + i\gamma\) for some real \(\alpha = \Re\rho\) and real \(\gamma = \operatorname{Im}(\rho)\).

Now we use the fact that \(J_{1/2}\) was constructed from the operator \(T_{1/2}\) whose kernel is \(K_{1/2}(\theta,\phi)=\sum_n\delta_n(\theta)\delta_n(\phi)/(n+2)\). The moments of \(\mu_{1/2}\) equal the traces \(\operatorname{tr}(\mathcal{L}_{1/2}^k)\), which by the determinant identity Theorem 9.1 and Corollary 9.2 count exactly the zeros of \(\zeta(\tfrac{1}{2}+it)\). Specifically, \(\mu_{1/2}\) is supported on the set \(\{t\in\mathbb{R} : \zeta(\tfrac{1}{2}+it)=0\}\) (by moment problem uniqueness). Hence \(\operatorname{spec}(J_{1/2})\subset\{t : \zeta(\tfrac{1}{2}+it)=0\}\).

But by the \(\sigma\)-independence argument (Theorem 12.4), the same set is the spectrum for all \(\sigma\), meaning every non‑trivial zero \(\rho\) (regardless of \(\Re\rho\)) contributes an element to \(\operatorname{spec}(J_{1/2})\). If \(\Re\rho\neq\tfrac{1}{2}\), say \(\Re\rho=\alpha\neq\tfrac{1}{2}\), then \(\rho\) contributes \(\operatorname{Im}(\rho)=\gamma\) to the spectrum; but \(\gamma\in\operatorname{spec}(J_{1/2})\) means \(\zeta(\tfrac{1}{2}+i\gamma)=0\). Together with the functional equation \(\zeta(\rho)=0\Leftrightarrow\zeta(1-\bar\rho)=0\), this forces \(\alpha=\tfrac{1}{2}\), a contradiction.

Therefore every non‑trivial zero satisfies \(\Re\rho=\tfrac{1}{2}\).
□

The proof reduces to the following chain of dependencies:

  ↓ super-exponential growth (Lemma 1.2)
Threshold angles & monotonicity (§2) → indicator structure of digits

Remaining formal tasks

The following results are used as axioms or with sorry in the Isabelle theories. Each has a reference to the literature or companion paper:

  1. Trace-class property of \(\mathcal{L}_s\) — adapts Mayer (1990); proof by singular value estimates.
  2. Similarity relation \(U_s\mathcal{L}_s=(M_{s+1/2}+K_s)U_s\) — finite branch-by-branch computation (Geere 2026).
  3. Telescoping trace sum (Lemma 8.1) — Markov-hole method; Efrat (1993), §3.
  4. Mayer–Efrat theorem (Theorem 7.1) — classical; Mayer (1990), Efrat (1993).
  5. Correlation kernel closed form (Theorem 3.3) — explicit integration with threshold angles (Geere 2026).
  6. Hausdorff moment problem uniqueness — classical measure theory (Hausdorff 1921).
  7. Spectrum of Jacobi matrix = support of \(\mu\) — standard spectral theory of Jacobi operators (Simon 2011).
