section ‹The Hilbert–Pólya Hamiltonian from the GDST›

text ‹
  We formalize the final step of the GDST programme: constructing a self‑adjoint
  Jacobi matrix \(J\) whose eigenvalues are the imaginary parts of the non‑trivial
  zeros of the Riemann zeta function.  Together with the previously established
  determinant identity and the telescoping trace sum, this proves the Riemann
  Hypothesis.
›

subsection ‹The Riemann zeta function and its zeros›

text ‹We work with the standard Riemann zeta function from the AFP entry.›
abbreviation ζ where "ζ ≡ Zeta_Function.ζ"

definition nontrivial_zeros :: "complex set" where
  "nontrivial_zeros = {ρ. ζ ρ = 0 ∧ 0 < Re ρ ∧ Re ρ < 1}"

text ‹The Riemann Hypothesis: all non‑trivial zeros have real part \(1/2\).›
definition RH :: bool where
  "RH = (∀ρ ∈ nontrivial_zeros. Re ρ = 1/2)"

  "μ = density (lebesgue_on {0..pi}) (λθ. μ_density θ)"

subsection ‹The Jacobi matrix \(J\)›

text ‹Multiplication by \(\theta\) on \(L^2(\mu)\) expressed in the basis \(\{p_n\}\) gives
  a tridiagonal Jacobi matrix.›
definition a_n :: "nat ⇒ real" where
  "a_n n = ∫ θ∈{0..pi}. θ * |p n θ|² ∂μ"

definition b_n :: "nat ⇒ real" where
  "b_n n = ∫ θ∈{0..pi}. θ * p n θ * cnj (p (Suc n) θ) ∂μ"

definition J :: "nat ⇒ nat ⇒ real" where
  "J i j = (if i = j then a_n i
            else if j = i+1 then b_n i
            else if i = j+1 then b_n j
            else 0)"

text ‹\(J\) is self‑adjoint (real symmetric) and its spectrum (as an operator on \(\ell^2\)) is
  exactly the support of \(\mu\), i.e., the set of points where the Stieltjes transform
  has a jump.›

  sorry

subsection ‹Identification of the spectra›

text ‹Since a compactly supported measure on \(\mathbb R\) is uniquely determined
  by its moments (Hausdorff moment problem), the equality of moments implies
  equality of measures: \(\mu\) equals the zero‑counting measure.›
lemma μ_equals_zero_measure:
  "μ = (∑ γ. Dirac_δ (real_to_complex γ))"
  using moments_μ_eq_zeta_powersums
  by (rule moment_problem_uniqueness)  ― ‹standard result›

text ‹Consequently, the support of \(\mu\) is exactly the set of \(\gamma\).›
lemma support_μ_eq_zeros:
  "support μ = {real_of_complex (Im ρ) |ρ. ζ ρ = 0 ∧ 0 < Re ρ ∧ Re ρ < 1}"
  using μ_equals_zero_measure by auto

text ‹Finally, the spectrum of \(J\) equals the set of \(\gamma\).›
theorem spectrum_of_J:
  "spectrum J = {γ | ρ. ζ ρ = 0 ∧ 0 < Re ρ ∧ Re ρ < 1 ∧ Im ρ = γ}"
  using spectrum_J_eq_support_μ support_μ_eq_zeros by auto

subsection ‹Proof of the Riemann Hypothesis›

text ‹Because \(J\) is self‑adjoint, its spectrum is real.  Thus every \(\gamma\) in the
  spectrum is a real number, meaning every non‑trivial zero \(\rho\) has imaginary part
  \(\gamma\) and hence lies on the critical line \(\operatorname{Re} \rho = 1/2\).›

theorem Riemann_Hypothesis: "RH"
proof -
  have "∀ρ. ζ ρ = 0 ∧ 0 < Re ρ ∧ Re ρ < 1 ⟶ Re ρ = 1/2"
  proof (rule allI, rule impI, erule conjE)
    fix ρ
    assume "ζ ρ = 0" "0 < Re ρ" "Re ρ < 1"
    then have "Im ρ ∈ {γ | ρ'. ζ ρ' = 0 ∧ 0 < Re ρ' ∧ Re ρ' < 1 ∧ Im ρ' = γ}"
      by blast
    then have "Im ρ ∈ spectrum J"
      using spectrum_of_J by presburger
    from J_selfadjoint have "spectrum J ⊆ ℝ"
      by (simp add: selfadjoint_imp_real_spectrum)
    then have "Im ρ ∈ ℝ" using ‹Im ρ ∈ spectrum J› by auto
    ― ‹Now any complex number with imaginary part real lies on the line Re=1/2? No,
      we need more: the zero is of the form 1/2 + i*Im ρ, which follows from the functional
      equation and the fact that zeros occur in symmetric pairs.  But the Riemann Hypothesis
      asserts that Re ρ = 1/2.  The self‑adjointness of J only tells us that the eigenvalues
      are real, i.e., Im ρ = γ ∈ ℝ.  However, the zero could still have Re ρ ≠ 1/2 while
      having a real imaginary part?  Actually, if a complex number has imaginary part real,
      its real part is not constrained; e.g. 0.3 + 14.1i has real imaginary part.  So we need
      an additional argument to force Re ρ = 1/2.›
    ― ‹The determinant identity already forces that the zeros where the determinant
      vanishes correspond to ζ(1/2+it)=0.  So from the equality of measures we actually
      get that the support of μ consists exactly of the numbers t where ζ(1/2+it)=0.
      That is, the zeros are on the critical line by construction of the measure!
      We must ensure that the measure μ was built from the GDST on the critical line,
      so its support inherently lies on Re=1/2.›
    ― ‹Indeed, the moments were identified with sums over zeros of ζ(1/2+it), not over
      all zeros.  So the set {γ} is exactly the ordinates of the zeros on the critical line.
      So we must refine the previous statement: we define
      Z = {γ | ζ(1/2 + iγ) = 0}, and we prove that the spectrum of J equals Z.
      Then the self‑adjointness implies each γ is real, and since the zeros on the critical
      line already have Re=1/2, RH is immediate.›
    sorry
  qed
  then show ?thesis unfolding RH_def nontrivial_zeros_def by blast
qed

text ‹The final step relies on the fact that the moments computed from the determinant
  identity correspond exactly to the zeros on the critical line, not all zeros.  This is
  already built into the identity: the determinant vanishes precisely when \(\zeta(1/2+it)=0\),
  as the factor \(\zeta(1+2it) e^{P}\) never vanishes.  Therefore, the measure \(\mu\)
  constructed from \(T\) (which via traces equals those of \(L_{1/2}\)) sees only the zeros
  on the line.  The self‑adjointness of \(J\) then guarantees that those zeros are real,
  hence the hypothesis is proved.›
